Instalasi Dan Konfigurasi Database MariaDB Di CentOS 7

Written by [Nur Hamim] with ‚ô• on July 6, 2019 in CENTOS 7 Administrator

Tahukan Anda tentang apa itu Database ?

Lalu buat apa sih fungsinya ?

Oke jika Anda penasang tentang pengertian fungsi database, berikut ini merupakan pengertian dan fungsi dari database .

Database Server merupakan sebuah program komputer yang menyediakan layanan pengolahan berbasis data dan melayani komputer atau program aplikasi basis data yang menggunakan model client ‚Äì server.

Fungsi Database Server :

  • Untuk menyimpan beberapa data yang bisa dilakukan dalam 1 lokasi
  • Penyimpanan yang aman karena fasilitas yang di server tidak ada di komputer biasa
  • Manajemen sistem
  • Mempermudah dan mecari dalam lingkup database

Setiap client dapat mengakses secara bersamaan

Database server sendiri bisa dibilang cukup banyak, yang biasanya sering digunakan yaitu MySQL, MariaDB dan PostgreSQL dan yang lainnya.

Nah pada tutorial kali ini saya akan share bagaimana cara instalasi dan konfigurasi database MariaDB di CentOS 7.

Pertama, silakan akses server Anda.

Kedua, jalankan baris perintah berikut

[root@srv-hamim01 ~]# yum install mariadb-server mariadb-client -y

Silakan tunggu proses instalasi sampai selesai jika sudah, selanjutnya Anda silakan start dan enable lalu melihat status dari database MariaDB telah berjalan dengan normal atau running. Fungsi enable sendiri berfunsi untuk menjalankan service secara otomatis apabila nantinya server dilakukan reboot atau restart, berikut perintah yang digunakan

[root@srv-hamim01 ~]# 
[root@srv-hamim01 ~]# systemctl start mariadb
