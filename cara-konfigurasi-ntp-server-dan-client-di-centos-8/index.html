Cara Konfigurasi NTP Server Dan Client Di CentOS 8

Written by [Nur Hamim] with ♥ on March 8, 2020 in CentOS 8 Administrator

NTP singkatan dari Network Time Protokol sebuah protokol yang digunakan untuk sinkronisasi waktu. NTP server sangatlah penting untuk diatur, jika Anda mempunyai sebuah aplikasi yang membutuhkan sinkronisasi waktu tertentu yang akurat, maka hal ini ini perlu di dikonfigurasi dengan baik dan benar.

Konfigurasi NTP Server

Melihat zona waktu yang digunakan oleh server Anda saat ini, gunakan perintah berikut

[root@hamim-centos8 ~]# timedatectl

Jika Anda ingin melihat list zona waktu yang tersedia, silakan gunakan perintah berikut

[root@hamim-centos8 ~]# timedatectl list-timezones

Namun jika Anda berada di Jakarta dan Anda ingin menyinkronisasi server Anda dengan waktu di Jakarta Anda dapat grep dari list diatas seperti berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# timedatectl list-timezones |grep Asia/Jakarta
Asia/Jakarta
[root@hamim-centos8 ~]#

Untuk set waktu nya dapat menggunakan perintah berikut, dan lihat perubahannya

[root@hamim-centos8 ~]# timedatectl set-timezone Asia/Jakarta
[root@hamim-centos8 ~]# timedatectl

Konfigurasi NTP Client

Tugas dari NTP Client sendiri yaitu melakukan sinkronisasi waktu dengan NTP Server, pastikan terlebih dahulu untuk NTP Client Anda sudah Aktif, gunakan perintah berikut:

[root@hamim-centos8 ~]# timedatectl | grep 'NTP service'
              NTP service: active
[root@hamim-centos8 ~]#

Jika NTP Client Anda belum Aktif silakan install service chrony terlebih dahulu berikut command yang digunakan untuk instalasinya

[root@hamim-centos8 ~]# dnf install chrony -y 
[root@hamim-centos8 ~]# systemctl enable chrony
[root@hamim-centos8 ~]# systemctl start chrony
[root@hamim-centos8 ~]# systemctl status chrony

Selanjutnya buka file chrony.conf yang berada di direktori /etc/chrony.conf silakan gunakan teks editor vi/vim/nano sesuai kenyamanan Anda

[root@hamim-centos8 ~]# vim /etc/chrony.conf

Disable pool 2.centos.pool.ntp.org iburst dan ubah NTP Server dengan lokasi Anda

Gunakan perintah berikut, untuk cek sources NTP Server

[root@hamim-centos8 ~]# chronyc sources

Untuk memeriksa sinkronisasi waktu sudah berjalan atau belum Anda dapat menggunakan tool NTPStat, silakan install terlebih dahulu menggunakan perintah

[root@hamim-centos8 ~]# dnf install ntpstat -y

Pastikan semua waktu mulai dari tanggal/bulan/tahun/ dan jam sudah sesuai semua dan berikut hasilnya

Selamat mencoba semoga bermanfaat.
