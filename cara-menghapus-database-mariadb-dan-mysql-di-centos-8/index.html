Cara Menghapus Database MariaDB dan MySQL Di CentOS 8

Written by [Nur Hamim] with â™¥ on March 17, 2020 in CentOS 8 Administrator

Pengenalan

Tutorial kali ini akan menjelaskan secara singkat bagaimana cara menghapus database MariaDB dan MsSQL di CentOS 8.

Salah satu tujuannya yaitu apabila Anda salah melakukan instalasi versi database yang Anda inginkan maka Anda tidak perlu rebuild server atau VM, Anda hanya cukup menghapus service database nya saja dengan mudah tentunya.

Menghapus Database MariaDB dan MySQL

Untuk menghapus database MariaDB Anda hanya perlu menjalankan satu baris perintah berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# dnf remove @mariadb -y

Untuk menghapus database MySQL Anda hanya perlu menjalankan perintah berikut

[root@hamim-centos8 ~]# 
[root@hamim-centos8 ~]# dnf remove @mysql -y

Langkah selanjutnya menghapus atau mengubah nama direktori MariaDB ataupun MySQL dimana secara default berada di /var/lib, namun apabila sudah Anda custome datadir dapat di lihat melalui direktori /etc/my.cnf.

Disini kami akan mengubah nama direktorinya saja supaya kita masih bisa menyimpan sebagai backup.

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# mv /var/lib/mysql /var/lib/mysql_mariadb_bkup
[root@hamim-centos8 ~]# 
[root@hamim-centos8 ~]# ls /var/lib/ |grep mysql
mysql_bkup
mysql_mariadb_bkup
[root@hamim-centos8 ~]

Sekian tutorial singkat menghapus database MariaDB dan MySQL di CentOS 8.
