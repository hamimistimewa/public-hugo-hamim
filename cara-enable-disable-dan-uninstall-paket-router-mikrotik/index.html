Karena semakin banyak paket â€“ paket yang kita gunakan maka semakin bertambah pula penggunaan memory pada mikrotik tersebut.

Disini saya akan mencontohkan cara Enable, Disable, dan Uninstall Paket Mikrotik. Sekarang kita akan mencoba Mendisable paket PPP dan MengEnable Paket IPv6.

Pertama kita bukak terlebih dahulu di menu System -> Packages lalu kita pilih paket atau fitur yang mau kita Enable atau Disable. Seperti pada gambar di bawah ini.

Apabila sudah di klik Enable atau Disable Packet tidak akan langsung Enable/Disable melainkan akan mendapatkan pemberitahuan scheduled yang berisi perintah apabila paket di Disable maka akan muncul scheduled for Disable . Untuk yang di Enable sama seperti yang Disable scheduled for Enable .

Setelah itu kita Reboot Router kita dengan cara masuk ke menu System -> Reboot

Setelah itu pilih Yes, tunggu Router sampai benar benar selesai reboot. Sebagai informasi tambahan apabila router sedang di reboot, jangan sampai router mati atau terlepas kabel powernya, karena hal tersebut akan akan membuat router bermasalah bahkan bisa sampai rusak.

Apabila router sudah reboot kita cek kembali apakah packet sudah enable atau disable

Sekarang packet IPv6 sudah tersedia, dan Packet PPP sudah di nonaktifkan.

Sebelumnya kita sudah belajar Enable dan Disable Packet Sekarang kita akan mencoba untuk menghapus atau uninstall packet di Mikrotik, dengan cara masuk ke menu system -> packages pilih paket yang ingin kita hilangkan atau di uninstall, setelah itu kita pilih uninstall seperti pada gambar di bawah ini.

Setelah itu kita reboot router kita dengan cara system -> reboot kemudian kita akan diminta untuk konfirmasi apakah anda ingin melanjutkannya atau tidak apabila anda indin melanjutkannya pilih Yes jika tidak pilih No
