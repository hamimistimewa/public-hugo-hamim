Instalasi Webserver Apache di CentOS 7

Written by [Nur Hamim] with ‚ô• on July 6, 2019 in CENTOS 7 Administrator

Pada kesempatan kali ini saya akan share bagaimana cara installasi webserver apache tersebut di sistem operasi CentOS 7 .

Untuk instalasi webserver apache kali ini dapat Anda coba di VPS (Virtual Private Server) atau VM (Virtual Machine) lokal biasanya banyak yang menggunakan Virtual Box di Komputer atau Laptop.

Pertama, silakan login terlebih dahulu ke server Anda masing ‚Äì masing,

Kedua, silakan lakukan pembaharuan atau update pada sistem operasi CentOS 7, dengan menggunakan perintah

[root@srv-hamim01 ~]# yum update -y

Silakan tunggu proses update sampai selesai yang bisanya membutuhkan waktu kurang lebih 10 ‚Äì 15 menit, tergantung dengan server yang kita gunakan.

Berikutnya, silakan menambahkan repository epel dengan cara menjalankan perintah

[root@srv-hamim01 ~]# yum install epel-release -y

Apabila sudah, selanjutnya Anda dapat memasang atau install webserver apache Anda dengan menjalankan satu baris perintah berikut

[root@srv-hamim01 ~]# yum install httpd -y

Tunggu proses instalasi selesai, jika sudah silakan start dan enable lalu lihat status webserver apache dengan cara berikut, fungsi enable disini berfungsi agar pada saat server mengalami reboot atau restart webserver kita tetap jalan atau running, berikut perintahnya :

[root@srv-hamim01 ~]# systemctl start httpd.service
[root@srv-hamim01 ~]# systemctl status httpd.service

Apabila webserver apache kita telah running dengan normal, silakan akses IP server Anda atau domain yang telah terpointing ke IP server Anda untuk memastikan apakah web server telah berjalan dengan normal.

Gambar diatas merupakan gambar atau tampilan default webserver apache di CentOS 7. Untuk testing kita dapat membuat file index.html di direktori /var/www/html/ seperti berikut untuk script sederhana file index.html-nya.

Oke, jika sudah silakan akses kembali IP atau domain Anda, maka akan tampil file index.html yang baru saja Anda buat.

Untuk melihat versi webserver apache yang kita gunakan dapat menggunkaan 2 perintah berikut diantaranya :

root@srv-hamim01 ~]# apachectl -v
Server version: Apache/2.4.6 (CentOS)
Server built: Nov 5 2018 01:47:09
[root@srv-hamim01 ~]#

Atau

[root@srv-hamim01 ~]# rpm -q httpd
httpd-2.4.6-88.el7.centos.x86_64
[root@srv-hamim01 ~]#

Itulah cara singkat untuk memasang atau install webserver apache di CentOS 7.
