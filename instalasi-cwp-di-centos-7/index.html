Instalasi CWP di CentOS 7

Written by [Nur Hamim] with â™¥ on July 6, 2019 in CONTROL PANEL

Mempunyai server VPS tapi nggak bisa mengelolanya, hemm jangan kebingungan dulu guyss, disini saya akan share bagaimana cara installasi control panel CWP (Control Web Panel) di sistem operasi CentOS 7 .

CWP (CentOS Web Panel) salah satu control panel yang powerfull yang memang dikhususkan untuk sistem operasi RHEL dan keturunannya yaitu CentOS, CWP juga dapat diinstal di CloudLinux.

CWP bersifat open source (free) bebas digunakan tanpa harus membeli license.

  • Ram 512MB lebih tinggi lebih baik dan direkomendasikan
  • Storage 10Gb lebih tinggi lebih baik dan direkomendasikan

Selanjutnya, saaya akan memeberitahukan tahapan â€“ tahapan untuk melakukan instalasi cwp di CentOS 7. Disini saya menggunakan VPS untuk servenya.

Pertama, pastikan VPS Anda dalam keadaan Freesh Install /OS atau belum terinstall aplikasi apapun.

Kedua, silakan remote VPS Anda, jika sudah silakan update terlebih dahulu VPS CentOS 7 Anda dengan menggunakan perintah yum update -y

Apabila sistem operasi CentOS 7 Anda telah ter-update dengan sempurna selanjutnya silakan menginstall service wget yang digunakan untuk mengunduh file installer dari cwp, berikut perintahnya

[root@srv-panel ~] # yum install wget -y

Jika sudah selanjutnya menentukan direktori installasi cwp, secara default direktori cwp yaitu /usr/local/src . Silakan pindah direktori terlebih dahulu menggunakan perintah cd , dan mengunduh file installer cwp seperti berikut perintahnya seperti berikut

[root@srv-panel ~]# cd /usr/local/src
[root@srv-panel src]# wget http://centos-webpanel.com/cwp-el7-latest

Jika sudah, selanjutnya Anda dapat melakukan instalasi cwp dengan menjalankan baris perintah berikut :

[root@srv-panel src]# sh cwp-el7-latest

Silakan tunggu proses instalasi sampai selesai, waktu instalasi membutuhkan waktu kurang lebih 1 jam.
