Konfigurasi FTP Server Di Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

Melanjutkan tutorial sebelumnya mengenai Instalasi FTP Server Di Windows Server 2016 pada tutorial kali ini kita akan membuat FTP dan bagaimana cara akses FTP yang berada di Windows Server 2016.

Berikut tahapan - tahapannya:

Silakan akses IIS Manager Windows Server 2016 >> Sites >> Add FTP Site

Pada menu FTP site name isikan nama FTP sesuai keinginan Anda. Untuk Physical path silakan tentukan direktori mana yang ingin di akses melalui FTP (sesuaikan dengan keinginan Anda)

Seperti berikut hasilnya. Next untuk melanjutkan ke tahapan berikutnya:

Selanjutnya, pada menu SSL silakan pilih yang No SSL karena untuk FTP kali ini kita tidak menggunakan SSL, klik Next

Selanjutnya, pada menu Authentication >> Basic dan untuk Authorization disini akan digunakan untuk All users untuk Permission yaitu Read and Write

Berikut tampilan dashboard hasil konfigurasi FTP Server Anda

Untuk memastikannya silakan buka CMD dan lakukan telnet ke port 21 sebagai berikut:

Apabila hasilnya seperti pada gambar dibawah ini maka FTP sudah dapat diakses dan digunakan

Selanjutnya untuk memastikannya, silakan akses di browser Anda masing - masing dengan cara mengetikan url seperti berikut: ftp://ip_server:21 lalu isikan username dan password login ke Windows Server Anda.

Jika berhasil akan seperti pada gambar di bawah ini

Sekarang kita coba akses menggunakan FTP Client, disini saya akan menggunakan Filezilla, berikut contoh konfigurasinya

Pada gambar di atas silakan dengan detail sebagai berikut:

  • Host: Isikan IP_Server Anda
  • Username: Username Windows Server Anda
  • Password: Password Windows Server Anda
  • Ports 21

Jika berhasil akan seperti gambar di bawah ini

Berikutnya, silakan mencoba melakukan transfer file dari lokal komputer ke Server Anda menggunakan Filezilla dengan cara drag and drop file atau direktori Anda, seperti pada gambar berikut

Selanjutnya silakan akses browser Anda sebelumnya, seharusnya file akan bertambah sesuai dengan file yang di transfer sebelumnya menggunakan FileZilla
