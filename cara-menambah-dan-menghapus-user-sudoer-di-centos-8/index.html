Cara Menambah dan Menghapus User (Sudoer) Di CentOS 8

Written by [Nur Hamim] with â™¥ on March 8, 2020 in CentOS 8 Administrator

Pengelanan User

Pada tutorial kali ini kami akan membahas tentang bagaimana cara membuat user dengan privileges root dan bagaiamana cara menghapus user tersebut pada CentOS 8.

Tujuan dari tutorial ini Anda dapat membuat user sesuai keinginan Anda, selain itu dari segi keamanan server juga sangat baik karena nantinya Anda dapat disable user root dan Anda dapat menggunakan user yang sudah dibuat dengan privileges root tersebut.

Tahapan Membuat Dan Menhapus User

Langkah pertama silakan remote akses terlebih dahulu server CentOS 8

Langkah selanjutnya gunakan beberapa perintah berikut untuk membuat user dan memberikan password pada user

[root@hamim-centos8 ~]# adduser hamim
[root@hamim-centos8 ~]# passwd hamim
Changing password for user hamim.
New password: 
Retype new password: 
passwd: all authentication tokens updated successfully.
[root@hamim-centos8 ~]#

Memasukan user hamim ke group wheel

[root@hamim-centos8 ~]# usermod -aG wheel hamim
[root@hamim-centos8 ~]# gpasswd -a hamim wheel 
Adding user hamim to group wheel
[root@hamim-centos8 ~]#

Langkah selanjutnya masuk ke user hamim menggunakan su -l dan pastikan user hamim udah berada di groups wheel berikut perintahnya

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# su - hamim
Last login: Sun Mar  8 12:55:30 WIB 2020 on pts/0
[hamim@hamim-centos8 ~]$ 
[hamim@hamim-centos8 ~]$ groups
hamim wheel
[hamim@hamim-centos8 ~]$

Verifikasi password root password dari user hamim menggunakan perintah sudo -l seperti berikut:

[hamim@hamim-centos8 ~]$ sudo -l

Hasilnya seperti pada gambar berikut

Mencoba login menggunakan user hamim dan memastikannya kembali, gunakan perintah berikut

[hamim@hamim-centos8 ~]$ sudo su
[root@hamim-centos8 hamim]# whoami
root
[root@hamim-centos8 hamim]#

Jika Anda ingin menhapus user hamim gunakan perintah berikut:

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# userdel -f -r hamim
[root@hamim-centos8 ~]#

Sekian tutorial terkait penambahan user dan delete user pada CentOS 8.
