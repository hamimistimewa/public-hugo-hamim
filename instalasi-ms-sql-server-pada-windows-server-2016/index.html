Instalasi MS SQL Server Pada Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

MS (Microsoft) SQL Server adalah sebuah sistem manajemen basis data relasional yang di peruntukan atau di rancang untuk aplikasi yang menggunakan arsitektur client - server.

Pada MS SQL Server terdapat sebuah fitur yang mempunyai sebuah kemampuan untuk membuat sebuah basis data baik itu mirroring dan clustering.

MS SQL Server juga mendukung SQL sebagai bahasa untuk memproses query ke dalam database dan kita tahu bahwa SQL ini sudah digunakan secara umum pada semua produk database server.

Perlu di ketahui untuk menggunakan MS SQL Server kita memerlukan sebuah License karena MS SQL tidak open source, untuk harga license nya sendiri bervariasi sesuai dengan edisi yang digunakan.

Sebagai contoh Anda dapat melihat pricing dari License MS SQL Server 2017 pada Link berikut: SQL Server pricing

Selengkapnya mengenai SQL Server dapat merujuk pada link berikut: Microsoft SQL Server

Berikut ini tahapan untuk instalasi MS SQL Server Pada Windows Server 2016.

Untuk MS SQL yang kita gunakan saat ini yaitu MS SQL Server 2017, berikut tampilan installasinya

Silakan pindah ke menu Installation , pada menu ini terdapat beberapa metode instalasi yang dapat Anda gunakan. Bila Anda ingin menerapkan MS SQL Server Failover Cluster, silakan pilih menu New SQL Server failover cluster installation namun bila posisinya stand-alone silakan pilih menu yang paling atas seperti pada gambar di bawah ini

Kumudian pada tahap ini Anda diminta untuk mengisikan License MS SQL Anda

Pada tahap ini silakan ceklis pada bagian (I Accept the license terms.) yang menunjukan Anda setuju semua tentang terms dari Lincense MS SQL Server Anda, klik Next

Pada menu Microsoft Update silakan Next

Tunggu setup file SQL Server sampai selesai

Instalasi Rules MS SQL Server sudah selesai, klik Next

Berikut ini merupakan tampilan default dari Feature Selection dimana pada menu Anda dapat memilih fitur apa saja yang ingin Anda gunakan dengan cara melakukan ceklist terhadap fitur yang tersedia. Selain itu pada menu ini Anda dapat menentukan patc root etc untuk instalasi MS SQL Anda. Klik next untuk lanjut ke tahapan berikutnya

Pada menu Feature Rules terdapat verifikasi atau rule yang akan di tentukan dari fitur yang ingin di install, pada gambar di bawah ini terlihat terdapat status error dikarenakan belum adanya Oracle JRE 7 (Java) yang terinstall di Windows Server 2016 Anda.

Berikut detailnya

Untuk mengatasi hal ini terdapat 2 cara yang dapat Anda gunakan.

  1. Intall terlebih dahulu Java JRE 7 sesuai rekomendasi
  2. Tidak melakukan ceklis pada fitur PolyBase (tidak di install)

Di sini kita akan menggunakan cara nomor 2 silakan back dan silakan non-ceklist pada menu fitur PolyBase Query Service for External Data , lalu Klik Next

Tunggu proses verifikasi rules selesai seperti pada gambar di bawah ini, kemudia klik Next

Pada tahap ini kita menggunakan default name Instance silakan klik Next

Pada tahap ini silakan klik Next

Pada tahap ini silakan pilih Windows authentication mode

Lalu Add Current User

Pada tahap ini silakan klik Add Current User lalu Next

Pada tahap ini silakan isikan nama controller sesuai dengan keinginan Anda

Selanjutnya klik pada Accept untuk install Microsoft R Open

Tunggu beberapa saat lalu Next

Selanjutnya klik pada Accept untuk install Python

Tunggu beberapa saat lalu Next

Pada tahap ini MS SQL Server siap di install silakan klik pada menu Install

Instalasi MS SQL sendiri membutuhkan waktu kurang lebih 10 - 45 menit. Silakan tunggu proses nya sampai selesai.

Jika proses instalasi telah selesai dan sukses maka hasilnya akan seperti pada gambar berikut
