Mengatasi Indexers Invalid Pada Magento 2

Written by [Nur Hamim] with ♥ on November 24, 2019 in Sinau Magento 2

Panduan kali ini saya akan membahas sedikit cara bagaimana cara troublehoosting bila mendapati kendala indexers invalid setelah melakukan instalasi Magento 2.2 seperti gambar diatas.

Untuk mengatasi kendala ini Anda dapat melakukan reindex melalui CLI command line di dalam direktori root Magento Anda dan melakukan clear cache pada Magento Anda sebagai berikut

[root@nur-srv-web02 ~]#
[root@nur-srv-web02 ~]# cd /usr/share/nginx/html/magento2/
[root@nur-srv-web02 magento2]# 
[root@nur-srv-web02 magento2]# php bin/magento indexer:reindex
[root@nur-srv-web02 magento2]# php bin/magento cache:flush

Jika sudah silakan akses kembali admin Magento Anda, hasil saat ini reindex sudah berhasil sebagai berikut hasilnya

Sekian panduan ini semoga bermanfaat bagi kita semua.
