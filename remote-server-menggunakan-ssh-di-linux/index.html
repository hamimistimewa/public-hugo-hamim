Remote Server Menggunakan SSH di Linux

Pernahkah Anda berfikir bagaimana sih cara yang dapat digunakan untuk mengakses sebuah server ?. Atau Anda yang telah mempunyai sebuah Virtual Private Server (VPS), tetapi bingung bagaimana cara mengaksesnya ?.

Oke, jangan keburu bingung dulu yaa, heheheh ðŸ™‚ , sebelum itu bagi Anda yang ingin mempunyai sebuah VPS yang murah dan handa serta sangat berkualitas bangett, saya sarankan Anda menggunakan VPS-nya CloudKilat, selengkapnya bisa di lihat pada tautan berikut : VPS CloudKilat

Oke, langsung saja yaa sebenarnya terdapat cara dan aplikasi yang dapat kita gunakan untuk mengakses sebuah server, jika Anda menggunakan sistem operasi Linux, Anda dapat mengakses sebuah server VPS tersebut melalui terminal linux dengan menggunakan ssh, lalu apasih itu ssh ?

Menurut wikipedia ssh adalah sebuah protokol jaringan yang aman serta berjalan di atas TCP/IP layer yang menggunakan port 22.

SSH sendiri tidak berjalan hanya di sistem operasi linux saja, ssh sekarang juga dapat dijalankan di sistem operasi sepertihalnya windows atau macos.

Selain itu saat ini sangat banyak aplikasi ssh yang berbasis GUI (Graphical User Interface), contohnya seperti Putty, Mputty dan masih banyak lagi yang lainnya.

Nah kali ini saya akan mencontohkan bagaimana cara remote atau mengakses sebuah server Virtual di menggunakan ssh dan terminal linux

Pertama, silakan akses terlebih dahulu terminal linux Anda

Kedua, silakan ketikkan perintah ssh root@IP_VPS_Anda , lalu konfirmasi authentikasi dengan mengetikkan perintah yes , dan isikan password VPS Anda seperti berikut

Oke saat ini Anda telah berhasil masuk/ login ke server VPS Anda, selamat mencoba semoga bermanfaat.
