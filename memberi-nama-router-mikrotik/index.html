Memberi Nama Router Mikrotik

Written by [Nur Hamim] with â™¥ on July 5, 2019 in NETWORKING

Pada kesempatan kali ini saya akan share bagaimana cara memberi nama pada mikrotik kita ok langsung saja yaa,, Pada lab kali ini kita akan memberi nama pada router kita, fungsi memberi nama pada router disini untuk mengetahui Mac Address dari router untuk login atau masuk di winbox. Ada 2 cara untuk memberi identitas atau nama pada router, pertama anda bisa menggunakan tools winbox, yang kedua bisa menggunakan terminal mikrotik dengan perintah : system identity set name=(Nama/Identitas Router).

Berikut ini cara memberikan identitas atau nama pada router mikrotik:

Menggunakan Winbox

  1. Pilih Menu System
  2. Identity
  3. Masukkan nama atau identitas

Menggunakan New Terminal di Mikrotik

  1. Masuk ke menu New Terminal
  2. Ketikkan perintah (system identity set name=(Nama/Identitas Router)
