Cara Menghubungkan Mikrotik Ke Access Point

Written by [Nur Hamim] with ♥ on July 6, 2019 in NETWORKING

Pada tutorial kali ini saya akan share bagaimana cara menghubungkan mikrotik ke access point. Berikut ini topologi sederhanan untuk menghubungkan perangkat keras Mikrotik Routherboard ke Internet, Untuk media Internetnya disini kita menggunakan Access Point, Mari kita mulai untuk mempraktekkannya :

Mikrotik memiliki berbagai macam jenis, dan banyak tipe – tipe mikrotik yang sudah memiliki fitur wireless, sepertihalnya RB941-2nD-TC yang sudah support wireless, kami akan mengkonfigurasikan mikrotik RB941-2nD-TC supaya terhubung ke internet/AP dengan MikroTik.

  • Langkah pertama kita buka aplikasi winbox, download langsung di situs resmi mikrtoik di https://mikrotik.com/download

  • Apabila sudah masuk, kita konfigurasikan alamat IP Address yang terhubung antara PC dan port Mikrotik. dengan perintah sebagai berikut : ip address add address 192.168.88.2⁄24 interface=ether2

  • Konfigurasi komputer kita agar satu network dengan interface Ether2, apabila ip ether2 adalah 192.168.88.2⁄24 (kelas c) maka ip komputer kita bisa menggunakan ip 192.168.88.3-192.168.88.254 dengan gateway 192.168.88.2 sesuaikan dengan alamat ip ether2 yang kita jadikan gateway. Disini kami akan memberikan cara untuk menyetting ip address pada sistem operasi windows dan linux ubuntu sebagai berikut :

Sistem Operasi Windows

Caranya masuk ke menu Network Station, Klik kanan di lokal area connection, lalu pilih propertie, Double klik di TCP/IP dan masukkan ip address serta gateway, lalu Ok.

Sistem Operasi Linux Ubuntu

Caranya Klik pada logo network pilih Edit connection, add, lalu pilih Create

Setelah itu kita kasih nama pada menu Connection name : “Untuk namanya bebas sesuai keinginan”, Pilih di menu Ethernet dan Di menu Device pilih Mac Addressnya

Selanjutnya kita masuk ke menu IPv4 Settings, Untuk Method, pilih Manual, klik Add dan isikan alamat IP Address, Netmask, Gateway, sesuaikan dengan Ip Address yang di Mikrotik, apabila sudah selesai kita simpan dengan klik tombol Save.

  • Selanjutnya kita aktifkan Fitur Wireless dengan klik menu Wireless -> Interface -> Wlan1 di enable/di aktifkan dengan cara (di centang) -> buka wirelessnya yang berada di menu interface klik tab wireless -> ganti band dengan 2GHz B/G/N -> Aplly -> Scan -> Pilih Access Point yang ingin digunakan -> lalu connect -> Aplly -> Ok

  • Apabila Access Point menggunakan Password, maka kita harus mengisi pasword tersebut degan benar, dan mengkonfigurasi Security Profile dengan cara: Wireless -> Security profile -> add

Centang semua Authentication Types dan Unicast Cipher serta Group Ciphers, isikan Password WPA Pre-Shared Key dan WPA2 Pre-Shared Key sesuai dengan Password Access Point anda.

  • Lalu kita masuk kembali ke manu Wireless, masukkan Security Profile yang barusan kita buat

Pastikan wlan1 Running, Seperti pada gambar dibawah ini :

  • Konfigurasi DHCP Client supaya bisa mendapatkan IP Address dari Acccess Point

  • Sekarang kita konfigurasikan NAT agar semua packet bisa di bungkus dan di sebarkan. Dengan perintah : IP -> Firewall -> NAT -> add: chain=srcnat, out interace wlan1 action = masquerade

  • Test apakah Mikrotik kita sudah mendapatkan koneksi internet. Caranya kita masuk ke Terminal Mikrotik, kita test dengan mengetikkan ping google.com. Apabila Mikrotik sudah mendapatkan koneksi internet baru kita test di komputer kita apakah sudah mendapatkan koneksi internet dengan cara mengetikkan perintah ping google.com pada cmd atau di terminal linux.

Selamat sekarang Anda sudah bisa mengkoneksikan Mikrotik ke Internet, Semoga bermanfaat, selamat mencoba 🙂
