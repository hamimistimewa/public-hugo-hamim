Pada saat Anda membuka vim pertama kali maka disitulah kita menggunakan yang namanya mode normal .

Mode normal ini dapat kita gunakan untuk navigasi (posisi atau arah) kursor serta juga dapat kita gunakan untuk berpindah antar file (apabila telah terdapat file).

Untuk memulai menulis sesuatu di Vim maka kita akan menggunakan mode insert , untuk masuk ke mode insert dari mode normal Anda perlu menekan huruf atau tombol i , perhatikan indikator di kiri bawah pada gambar berikut

Pada Mode Insert ini Anda dapat menulis atau mencatat apa yang Anda inginkan, sebagai contoh penulis disini akan menulis beberapa kalimat seperti berikut

Untuk keluar dari mode insert tersebut silakan tekan tombol Esc maka Anda akan kembali ke mode normal .

Berikutnya adalah mode command atau perintah, mode ini ditandai dengan kita mengetikkan tombol : dan kemudian diikuti dengan perintah yang ingin dijalankan, misalnya untuk menyimpan file pada gambar diatas dapat dilakukan dengan cara menekan huruf w dan diikuti nama file yang ingin disimpan misalnya pada file ini kita akan memberikan nama file belajar.txt kemudian silakan tekan tombol Enter untuk meng-eksekusinya, contoh seperti pada gambar di bawah ini

Selain itu mode command juga dapat kita gunakan untuk keluar dari Vim dengan cara tekan tombol : lalu huruf q seperti pada gambar berikut

Silakan lihat file belajar.txt yang baru saja Anda buat dengan cara menjalankan perintah ls seperti berikut

Gambar diatas menandakan kita telah berhasil untuk membuat file dan menyimpan file menggunakan Vim. Sekarang silakan buka kembali Vim Anda, disini kita akan mempelajari bagaimana cara membuka file yang sudah ada di Vim.

Untuk membuka file yang sudah ada tersebut dapat dilakukan dengan cara menjalankan perintah : dan menekan tombol e (:e) diikut dengan nama file Anda. Disini file yang dimaksud yaitu belajar.txt (silakan sesuaikan dengan file yang ada), lalu tekan Enter untuk meng-eksekusinya, dengan itu file belajar.txt akan terbuka, seperti pada gambar berikut
