Mengenal Editor Vim

Written by [Nur Hamim] with ♥ on July 6, 2019 in TEKS EDITOR

Jika Anda seorang programmer atau seorang sysadmin atau Anda pengguna sistem operasi Linux atapun Windows maka Anda akan kenal dan tak akan asing lagi dengan yang namanya teks editor sebagai contoh teks editor yang populer dan biasa dipakai bagi pengguna siste operasi Windows yaitu “notepad dan notepad++” . Selain itu bagi pengguna sistem operasi Linux pasti akan kenal pakai (banget) dengan yang namanya teks editor nano, terlebih bagi pengguna Linux Ubuntu.

Selain itu sebenarnya sangat banyak pemilihan teks editor yang dapat kita gunakan diantaranya :

  • Nano
  • Sublime Teks
  • Visual Studio Code
  • Notepad++
  • Kate
  • Atom

Mungkin diatas beberapa teks editor yang dapat penulis sampaikan, namun sebenarnya masih sangat banyak yang dapat Anda gunakan.

Pada kali ini penulis akan memperkenalkan sebuah teks editor yang bisa dibilang magic dan powerfull dan sebagian orang terkadang menyebutnya editor yang paling susah digunakan. wawww bisa gitu yaa hihihi 🙂

Teks editor Vim adalah teks editor yang sangat powerfull dan mudah digunakan. Namun banyak juga yang beranggapan bahwasannya VIM itu susah , Yaps betul karena vim salah satu teks editor yang berbeda dengan teks editor yang lainnya . Dengan menggunakan editor vim kita dapat berselancar dengan mudah tanpa ketergantungan dengan yang namanya tikus (mouse) hihihi 🙂

Editor Vim sendiri memiliki berbagai macam pemilihan penggunaan mulai dari GUI (Graphic User Interface) sampai ke CLI (Command Line Interface) . Namun pada kesempatan kali ini penulis akan membahas penggunaan Vim menggunakan CLI.

Sebelum melanjutkan ke pembahasan – pembahasan selanjutnya mengenai Vim ada baiknya bagi Anda yang belum memasang atau install Vim, berikut merupakan langkah – langkah untuk instalasi Vim di sistem operasi Linux

Terlebih dahulu silakan update sistem operasi Ubuntu yang Anda gunakan disini penulis menggunakan Ubuntu 18.04, berikut perintahnya :

$ sudo apt update

Jika sudah berikutnya Anda dapat memasang editor Vim dengan menjalankan satu baris perintah berikut

$ sudo apt install vim

Untuk melihat versi dari Vim dapat menjalankan perintah berikut

$ vi --version

Jika sudah Anda dapat membuka Vim dengan mengeksekusi perintah vim di terminal linux Anda,

$ sudo vim

Maka akan muncul tampilan default Vim seperti gambar dibawah ini

Oke, saat ini editor Vim telah berhasil diinstall dan telah dapat dibuka dengan baik.

Selamat mencoba semoga bermanfaat 🙂
