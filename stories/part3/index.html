My First eBook

Written by [Nur Hamim] with ♥ on July 12, 2019 in

#Cah Ndeso

Yah saya anak desa tepatnya beralamat di Dsn. Talian, Ds. Panyepen, Kec. Jrengik, Kab. Sampang Madura Jawa Timur.

Tepat pada hari senin tanggal 16 September 2019 saya bisa menyelesaikan eBook pertama saya mengenai CentOS 7 Linux Administration Pada Cloud Computing.

Pada saat lulus SMK Alhamdulillah saya belum di takdirkan untuk kuliah, namun sang khalik berkata lain saya berkesempatan #Nyantri IT di Semarang tepat nya Santren Koding (santrenkoding.org). Terima kasih banyak saya ucapkan kepada Bapak Ariasetia Alam yang telah mendidik saya selama di Santren.

Lanjut cerita saya berkesempatan untuk mengikuti Bootcamp Linux Administration yang diadakan Komunitas GNU/Linux Malang (klim.or.id) dimana pada saat itu juga #Alhamdulillah saya bisa berkesempatan untuk memulai karir di dunia IT Cloud Computing sampai saat ini.

Pada eBook ini banyak materi dasar dan menengah terkait Linux CentOS dimana semua yang ada di eBook merupakan hasil rangkuman belajar.

Saya ucapkan terima kasih banyak kepada Om Wildan Zubaidi dan Om Febri Nur Aswin yang telah dan mau mendidik saya cah Ndeso ini.

eBook ini saya persembahkan untuk teman - teman komunitas Linux tentunya yang ingin belajar terkait CentOS 7 Linux Administration secara [Free]

Saya meminta dan memohon maaf bila terdapat penulisan yang kurang berkenan atau pun typo karena saya sendiri masih tahap belajar untuk lebih baik lagi, kritik saudara dan teman - teman sangat membantu saya.

Untuk mengunduh eBook silakan klik pada link berikut: CentOS 7 Linux Administration Pada Cloud Computing

Salam Hangat
