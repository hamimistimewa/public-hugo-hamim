Cara Mengaktifkan Telnet Di Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

Telnet salah satu service yang dapat kita gunakan untuk mengetahui atau melihat sebuah service dalam ke adaan open atau running tidak nya sebuah service dengan menggunakan ip ataupun domain yang telah terpointing ke ip tersebut.

Pada gambar di atas terlihat service telnet di Windows Server 2016 belum dapat digunakan, oleh karena itu kita perlu mengaktifkannya.

Berikut tahapan - tahapannya:

Silakan masuk ke Server Manager >> Add roles and features

Kemudian Next

Pada Installation Type silakan pilih Roles-based or feture-based installation lalu Next

Pilih server Anda kemudian Next

Pada Server Roles pastikan File and Storage Services sudah terinstall, kemudian Next

Pada menu Features silakan pilih dan centang fitur Telnet Client & TFTP Client

Menjadi seperti pada gambar dibawah ini, lalu Next

Install service Telnet

Tunggu proses instalasi sampai selesai

Instalasi Telnet telah berhasil di lakukan

Untuk memastikannya silakan masuk ke Command Prompt

Ketikan perintah telnet ip_server 80 atau telnet google.com 80

Berikut contoh bila menggunakan powershell di Windows Server 2016
