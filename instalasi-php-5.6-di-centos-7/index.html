Pertama, silakan login ke server Anda masing â€“ masing.

Kedua, silakan update sistem operasi CentOS 7 Anda dengan menjalankan perintah berikut

[root@srv-hamim01 ~]# yum update -y

Jika sudah silakan, berikutnya menambahkan atau install repositori remi di CentOS 7 dengan menjalankan perintah

[root@srv-hamim01 ~]# yum install http://rpms.remirepo.net/enterprise/remi-release-7.rpm

Jika sudah berikutnya kita install manager untuk repositori remi di CentOS 7 dengan menggunakan perintah berikut

Jika sudah, berikutnya install php yang dibutuhkan,

[root@srv-hamim01 ~]# yum install php php-mcrypt php-cli php-gd php-curl php-mysql php-ldap php-zip php-fileinfo

Silakan tunggu proses installasi php sampai selesai. Untuk melihat versi php yang kita gunakan dapat dilakukan dengan menggunakan perintah

[root@srv-hamim01 ~]# php -v
PHP 5.6.40 (cli) (built: Jan 9 2019 12:34:16)
Copyright (c) 1997-2016 The PHP Group
Zend Engine v2.6.0, Copyright (c) 1998-2016 Zend Technologies
[root@srv-hamim01 ~]#

Berikutnya mari kita cek php apa saja yang telah kita install dengan membuat file info.php di /var/www/html dengan cara berikut,

Pindah ke direktori /var/www/html menggunakan perintah cd .

Buat script info.php seperti berikut menggunakan teks edito vim atau nano,

Jika sudah silakan simpan file info.php tersebut, selanjutnya melakukan restart atau reload pada webserver yang Anda gunakan disini menggunakan web server apache, berikut command yang dapat Anda gunakan untuk melakukan reload tersebut

[root@srv-hamim01 html]# systemctl reload httpd

Jika sudah silakan akses IP server Anda dengan /info.php . Maka hasilnya akan seperti pada gambar dibawah ini

Pada tahap ini installasi php di CentOS 7 sudah selesai cukup mudah kan.
