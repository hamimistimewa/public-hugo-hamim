Cara Menambahkan Website/Virtual Host Di Web Server IIS Pada Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

Tutorial kali merupakan kelanjutan dari tutorial sebelumnya.

Bila pada tutorial sebelumnya yaitu mengenai instalasi Web Server IIS(https://nurhamim.net/mengenal-dan-instalasi-web-server-iis-di-windows-server-2016/ “”) , maka untuk tutorial kali merupakan bagaimana cara menambahkan Website atau Virtual Host di Web Server IIS.

Silakan akses Windows Server 2016 Anda terlebih dahulu menggunakan RDP

Buka IIS Manager di menu Tools >> Internet Information Services (IIS) Manager.

Tampilan Manager Web Server IIS

Untuk menambahkan site/virtualhost , drop-down HAMIM-WINDOWS >> SITES >> klik kanan >> Add Website

Pada tampilan di bawah ini, isikan sesuai parameter website Anda masing - masing.

Berikut detailnya:

  • Site Name: Isikan dengan nama domain yang Anda digunakan
  • Physical Path: Pilih Direktori website Anda
  • Hostname: Isikan dengan nama domain yang Anda digunakan
