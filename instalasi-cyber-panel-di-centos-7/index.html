Tutorial kali ini temanya masih sama dengan postingan yang sebelumnya yaitu mengenai instalasi control panel, namun kali ini kita akan mencoba control panel Cyber Panel di CentOS 7.

Control Panel Cyber Panel bisa dibilang control panel yang open source (bebas) yang tergolong baru, dan sangat banyak fitur â€“ fitur keren yang dapat kita manfaatkan untuk mengelola sebuah website.

Adapun fitur â€“ fitur yang ada di cyber panel diantaranya, DNS Management, Mail, FirewallD and SpamAssassin untuk security, SSL Free, dan webserver menggunakan OpenLiteSpeed, dimana penggunaan dan cache website Anda akan terasa ringan dan cepat, terlebih apabila Anda menggunakan CMS WordPress pada control panel ini.

Untuk lebih detail mengenai fitur yang berada di Cyber Panel dapat dilihat disini : Fitur Cyber Panel

Sebelum melakukan instalasi control panel Cyber Panel pastikan sistem operasi CentOS 7 Anda bersifat Fress OS atau belum terinstall aplikasi apapun didalamnya. Oke langsung saja kita ke langkah â€“ langkah instalasi Cyber Panel di CentOS 7 :

Pertama, silakan remote server atau VPS Anda terlebih dahulu menggunakan ssh ataupun yang lainnya.

Kedua, silakan update sistem operasi CentOS 7 Anda dengan menggunakan perintah :

[root@srv-panel ~] # yum update -y

Silakan tunggu pembaharuan atau udate sistem selesai, jika sudah selanjutnya menginstall service wget nantinya digunakan untuk mengunduh file control panel cyber panelnya, seperti berikut perintahnya

[root@srv-panel ~] # yum install wget -y

Jika sudah, silakan unduh dan install control panel Cyber Panel dengan menjalankan 1 bari perintah berikut :

[root@srv-panel ~] # sh <(curl https://cyberpanel.net/install.sh || wget -O - https://cyberpanel.net/install.sh)

Gambar di atas silakan pilih nomor untuk installasi Cyber Panel, Selanjutnya Anda akan diminta untuk memilih versi webserver OpenLiteSpeed yang ingin digunakan di control panel Cyber Panel, terdapat 2 pilihan yaiu Free dan berbayar, jika Anda ingin menggunakan yang berbayar silakan pilih nomor 2, jika Anda ingin menggunakan yang free silakan pilih yang nomor 1 seperti pada gambar dibawah ini

Selanjutnya, Anda akan diminta untuk memilih penggunaan password admin, apakah Anda ingin menggunakan password random yang nantinya akan ditentukan oleh sistem atau manual, jika random silakan tekan huruf r .

Berikutnya Anda diminta untuk melakukan konfirmasinya extension php yang ingin Anda gunakan, ini bersifat opsional yaa, jika Anda tidak membutuhkan extension tersebut silakan skip saja, seperti berikut contohnya

Silakan tunggu proses instalasi sampai selesai, dimana proses instalasi membutuhkan waktu kurang lebih 15 â€“ 40 menit. Apabila instalasi telah selesai maka Anda akan diberikan credential untuk login ke admin control panel Cyber Panel beserta credential untuk login ke admin Database, contohnya seperti pada gambar dibawah ini.

Silakan akses control panel Cyber Panel Anda dengan menggunakan port 8090 . contohnya https://IPSerever:8090, seperti pada gambar dibawah ini

Silakan isikan username dan password untuk login ke admin panel Anda seperti pada gambar diatas, apabila credential login telah berhasil dan benar maka Anda akan diarahkan ke dashboard admin Cyber Panel yang cukup bagus dan sangat enak dilihat heehehhee ðŸ™‚
