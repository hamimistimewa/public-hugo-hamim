Instalasi SQL Server Management Studio (SSMS) Pada Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

Tutorial kali ini merupakan kelanjutan dari tutorial sebelumnya mengenai Instalasi Microsoft SQL Server 2017 pada Windows Server 2016, selengkapnya mengenai instalasi dapat merujuk pada link berikut:Instalasi MS SQL Server Pada Windows Server 2016

SQL Server Management Studio (SSMS) merupakan tools yang dapat digunakan untuk mengatur segara infrastruktur SQL mulai dari SQL Server ke SQL Database selain itu dapat digunakan untuk melakukan konfigurasi, memantau (monitoring), dan mengelola SQL baik itu melakukan deploy, upgrade komponen data-tier yang digunakan oleh aplikasi (termasuk membuat query dan script).

Selengkapnya dapat merujuk pada link berikut: What is SQL Server Management Studio (SSMS)?

Berikut ini tahapan instalasinya.

Pertama silakan klik pada menu Install SQL Server Management Tools

Selanjutnya Anda diarahkan untuk mengunduh SQL Management Studio (SSMS) seperti pada gambar berikut ini

Atau Anda dapat mengunduh secara langsung melalui link berikut ini: Download SSMS 18.2

Kemudian klik pada file .exe yang telah Anda unduh sebelumnya

Pilih Run untuk melanjutkan proses instalasi

Default path SSMS berada di **C:\Program Files (x86)Microsoft SQL Server Management Studio 18 seperti pada gambar di bawah ini

Bila Anda ingin custome path direktorinya dapat dilakukan dengan cara klik pada menu Change , lalu pilih Direktori sesuai keinginan Anda, seperti pada gambar di bawah ini

Untuk melanjutkan proses installasi silakan klik menu Install

Silakan tunggu proses instalasi sampai selesai

Apabila sudah selesai, maka Anda diminta untuk melakukan restart seperti gambar berikut

Saat ini instalasi telah selesai dilakukan, selanjutnya silakan akses SSMS seperti berikut

Berikut tampilan default dari Microsoft SQL Server Management Studio , untuk loginnya silakan isikan Server-Name Windows Server Anda lalu klik Connect

Saat ini MS SQL Anda sudah dapat diakses menggunakan management tools SSMS
