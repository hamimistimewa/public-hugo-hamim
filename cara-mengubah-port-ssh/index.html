Cara Mengubah Port SSH

Written by [Nur Hamim] with â™¥ on July 6, 2019 in CENTOS 7 Administrator

Pada tutorial sebelumnya saya telah menyampaikan bahwa secara default ssh menggunakan port 22, nah pada tutorial kali ini kita akan mempelajari bagaimana cara untuk memodifikasi atau mengubah port default ssh tersebut.

Sangat banyak manfaat yang didaptkan jika kita mengubah port default ssh salah satunya adalah melindungi service ssh kita dari serangan brutforce ssh yang sangat sering sekali terjadi.

Namun terdapat juga kekurangannya yaitu jikalau kita lupa port ssh yang telah kita ubah sebelumnya yaa mau nggak mau kita tidak akan bisa login ke server ğŸ™

Oke, berikut ini langkah â€“ langkah untuk mengubah port ssh,

Pertama, silakan masuk/ login ke server Anda terlebih dahulu

Kedua, silakan buka konfigurasi ssh yang berada di direktori sshd_config, seperti berikut :

[root@srv-hamim01 ~]# vim /etc/ssh/sshd_config

Disini kami menggunakan CentOS 7 dengan demikian untuk mengubah konfigurasinya bisa menggunakan editor vi atau vim atau bisa juga dengan nano . Jika sudah silakan cari port 22 seperti gambar dibawah ini

Ubah port 22 diatas sesuai dengan keinginan Anda, misalnya disini kami menggunakan port 22222,

Jika sudah silakan simpan dan silakan restart service ssh Anda dengan menggunakan perintah systemctl restart sshd.service, seperti pada gambar dibawah ini

Ketiga, tahap pengujian silakan remote kembali ssh Anda secara manual tanpa port apapun seperti langkah pertama, hasilnya Anda tidak akan bisa login ke server

Selanjutnya, coba login kembali dengan memberikan akhiran -p [port ssh] seperti pada gambar dibawah ini
