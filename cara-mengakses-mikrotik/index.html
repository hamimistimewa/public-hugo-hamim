Cara Mengakses Mikrotik

Written by [Nur Hamim] with ♥ on July 5, 2019 in NETWORKING

Kali ini saya akan share bagaimana cara dan menggunakan aplikasi atau bagaimana supaya kita dapat mengakses Router Mikrotik ?

Langsung saja ya untuk mengakses router pertama kali, ada banyak cara yang bisa kita gunakan, berikut ini beberapa cara mengakses router, diantaranya adalah:

Untuk mengakses Mikrotik Routherboard ataupun Mikrotik RouterOS yang baru, atau yang sudah tereset, memiliki default konfigurasi IP pada Ether1 yaitu 192.168.88.1 / 24, dengan default username yaitu Username: admin, dan Password: (Kosong).

Ada 4 cara untuk megakses MikroTik Router yang biasa atau umumnya digunakan untuk mengakses yaitu:

Via Console/Command Mikrotik

Kita bisa mengakses Mikrotik menggunakan console atau shell ataupun remote akses menggunakan Applikasi PUTTY (http://www.putty.org/). Applikasi Putty ini bisa Anda install dan digunakan di MSI (‘Windows Installer’) dan Unix source archive. Sekarang kita coba akses Mikrotik dengan Putty, caranya tinggal masukkan alamat IP Mikrotik ke kolom Host Name nya Putty

Maka akan tampil seperti pada gambar dibawah ini

Perlu Anda ketahui apabila anda meremote Mikrotik menggunakan Putty Anda harus memasukkan perintah atau command.

Berikut ini tips supaya kita lebih gampang, dan menghemat waktu, tenaga serta lebih efisien megurangi kesalahan-kesalahan, untuk mengkonfigurasi Mikrotik menggunakan Command.

Tipsnya yaitu “Memanfaatkan auto complete” (seperti bash auto complete di linux) Tekan Tombol Tab di pada keyboard untuk mengetahui atau melengkapi daftar perintah selanjutnya. sebagai contoh jika Anda ingin masuk ke menu IP Address kita tinggal ketikkan perintah ip add > lalu tekan TAB > maka otomatis shell akan melengkapinya menjadi Ip Address. Untuk kembali ke sub mneu ketikkan perintah “ . . ” (titik dua), Untuk kembali ke menu root ketikkan perintah “ / ”

Via Web Browser

Apabila kita tidak mempunyai aplikasi pendukung mikrotik sama sekali, Anda tidak perlu berkecil hati karena mikrotik juga bisa anda remote via web browser dengan port 80 pada bworser.Contoh : Ketikkan ip mikrotik kita di browser, disini ip mikrotik kami 192.168.88.2

Isikan Usernama (Login)=(disesuaikan) default dari Mikrotik yaitu admin, setelah itu masukkan Password (sesuaikan dengan password Router) default password Mikrotik (kosong)

Cara ini merupakan cara yang paling mudah, dari pada kita harus menggunakan aplikasi pendukung yang biasanya akan terjadi masalah atau error yng lebih banyak dari aplikasinya tersebut.

Via Winbox

Apa itu Winbox ? Winbox merupakan tools untuk me-remote Mikrotik yang paling populer dan selalu up to date, winbox (utility kecil di windows yang sangat praktis dan cukup mudah diguankan), winbox juga bisa berjalan di linux bisa memakai Wine untuk menjalankannya bisa juga langsung di install di sistem operasi tersebut. Winbox salah satu tools yang bersifat GUI (Grafik User Interface). Berikut ini tampilan awal dari winbox:

Winbox akan mendeteksi mikrotik yang sudah di install atau sudah terkoneksi ke laptop atau komputer asal masih dalam satu network atau jaringan, biasanya Winbox akan mendeteksi alamat Mac Address dari ethernet yang terpasang di mikrotik. Kita bisa mengakses Mikrotik menggunakan winbox dengan menggunakan alamat IP Address Mikrotik maupun Mac Asdressnya.

Via Telnet

Mikrotik juga dapat kita remote menggunakan Telnet melalui program aplikasi “command prompt” (cmd) yang sudah tersedia di sistem operasi windows, Tapi tidak direkomendasikan menggunakan Telnet dalam jaringan karena masalah keamanannya.
