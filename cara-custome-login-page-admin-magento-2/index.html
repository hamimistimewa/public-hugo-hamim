Cara Custome Login Page Admin Magento 2

Written by [Nur Hamim] with ♥ on November 24, 2019 in Sinau Magento 2

Panduan kali ini saya akan berbagi bagaimana cara mengubah atau custome login page Admin dari Magento 2.2.

Untuk custome login page ini silakan login ke server web Magento Anda terlebih dahulu dan pindah ke direktori root Magento seperti berikut:

[root@nur-srv-web02 ~]# 
[root@nur-srv-web02 ~]# cd /usr/share/nginx/html/magento2/
[root@nur-srv-web02 magento2]#

Kemudian, pindah ke direktori /app/etc/ dan buka file env.php.

[root@nur-srv-web02 magento2]# cd app/etc/
[root@nur-srv-web02 etc]# 
[root@nur-srv-web02 etc]# vim env.php

Ubah bagian 'frontName' pada backend seperti berikut ini

'frontName' => 'admin_4n57aw' >> 'frontName' => 'Hamimaja'

Disini saya mengubah menjadi Hamimaja

Jika sudah silakan simpan perubahan yang telah dilakukan.

Selanjutnya silakan restart service Nginx dan php-fpm, serta flush cache magento menggunakan perintah berikut

[root@nur-srv-web02 magento2]#
[root@nur-srv-web02 magento2]# systemctl restart nginx && systemctl restart php-fpm
[root@nur-srv-web02 magento2]#
[root@nur-srv-web02 magento2]# php bin/magento cache:flush

Selanjutnya silakan akses kembali hamim.web.id/Hamimaja untuk mengakses admin page Magento Anda hasil nya akan sebagai berikut

Sekian panduan terkait custome login page Admin pada Magento 2. Semoga bermanfaat bagi kita semua.
