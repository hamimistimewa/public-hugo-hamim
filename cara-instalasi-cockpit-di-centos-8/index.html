Cara Instalasi Cockpit Di CentOS 8

Written by [Nur Hamim] with â™¥ on March 8, 2020 in CentOS 8 Administrator

Pengenalan

Cockpit adalah tools yang digunakan untuk manajemen server berbasis web. Secara default pada CentOS 8 tools Cockpit sudah ada di dalam repository.

Cockpit dapat digunakan oleh sysadmin untuk memantau, mengelola sepertihalnya network, storage, container, dan sistem log dari aplikasi yang terinstall di Server.

Instalasi Dan Penggunaan Cockpit

Untuk instalasi cockpit pada CentOS 8 Anda dapat menjalankan baris perintah berikut:

[root@srv-centos8 ~]# dnf install cockpit -y

Jika sudah silakan start dan enable service cockpit, gunakan perintah berikut:

[root@srv-centos8 ~]# systemctl start cockpit.socket
[root@srv-centos8 ~]# systemctl enable cockpit.socket

Pastikan service cockpit sudah running gunakan perintah berikut

[root@srv-centos8 ~]# systemctl status cockpit.socket
[root@srv-centos8 ~]# ss -tunlp |grep cockpit
[root@srv-centos8 ~]# ps auxf|grep cockpit

Akses IP_Public atau IP_Server Anda menggunakan port 9090 karena cockpit listen di port tersebut, sebagai berikut

Berikut tampilan dari Dashboard Cockpit

Untuk melihat log server server silakan masuk ke menu Logs

Untuk melihat bandwith network ataupun jika ingin menambahkan interface network dapat dilakukan pada menu Networking

Untuk melihat akun atau menambahkan akun dapat melalui menu Accounts

Untuk melihat Service yang aktif ataupun tidak dapat dilihat pada menu Services

Jika Anda ingin mengelola SELinux pada menu

Jika Anda ingin menggunakan terminal untuk pengoperasian server by CLI juga dapat dilakukan silakan ke menu Terminal
