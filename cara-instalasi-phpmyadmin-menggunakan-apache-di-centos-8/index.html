Cara Instalasi phpMyAdmin Menggunakan Apache Di CentOS 8

Written by [Nur Hamim] with â™¥ on March 15, 2020 in CentOS 8 Administrator

Pengenalan

phpMyAdmin merupakan aplikasi open source yang ditulis menggunakan PHP untuk mengelola server database MySQL dan MariaDB melalui antarmuka Web.

phpMyAdmin memiliki dukungan untuk berbagai operasi di MySQL, MariaDB, dan Drizzle. Dengan phpMyAdmin, Anda dapat mengelola database, tables, columns, relations, indexes, useres, permissions, dan lainnya melalui antarmuka web yang intuitif dan mudah digunakan.

Instalasi phpMyAdmin

Untuk mengikuti tutorial ini pastikan Anda telah melakukan instalasi apache web server dan PHP, untuk panduannya dapat mengikuti link berikut:

  • Cara Instalasi Web Server Apache Di CentOS 8
  • Cara Instalasi PHP 7.2, 7.3 dan 7.4 Apache Di CentOS 8

Dan pastikan Anda sudah melakukan instalasi database MySQL ataupun MariaDB, berikut panduan untuk instalasi MariaDB

  • Cara Instalasi Database MariaDB Di CentOS 8

Langkah selanjutnya pastikan untuk module php-mysqlnd sudah terinstall

[root@hamim-centos8 ~]# 
[root@hamim-centos8 ~]# php -m |grep mysqlnd
mysqlnd
[root@hamim-centos8 ~]#

Jika sudah silakan unduh phpMyAdmin dapat melalui link berikut: Klik DISINI

Atau Anda dapat menggunakan perintah berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# wget https://files.phpmyadmin.net/phpMyAdmin/5.0.1/phpMyAdmin-5.0.1-all-languages.zip

Selanjutnya unzip file phpMyAdmin

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# unzip phpMyAdmin-5.0.1-all-languages.zip

Jika sudah ubah nama file phpMyAdmin, untuk penamaan silakan sesuaikan dengan keinginan

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# mv phpMyAdmin-5.0.1-all-languages phpmyadmin

Pindah file phpmyadmin ke direktori /var/www/

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# mv phpmyadmin/ /var/www/

Change owner menjadi apache untuk direktori phpmyamdin

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# chown -Rf apache:apache /var/www/phpmyadmin/

Kemudian, membuat vhost untuk phpmyadmin

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# vim /etc/httpd/conf.d/phpmyadmin.conf

Berikut isi file vhost phpmyadmin.conf

Alias /phpmyadmin /var/www/phpmyadmin

Langkah selanjutnya konfigurasi phpmyadmin, silakan masuk ke root database Anda dan membuat username dan password untuk login ke phpmyadmin

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# mysql -u root -p
Enter password:
Welcome to the MariaDB monitor.  Commands end with ; or \g.
Your MariaDB connection id is 17
Server version: 10.3.17-MariaDB MariaDB Server

Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.

MariaDB [(none)]>
MariaDB [(none)]> GRANT ALL ON *.* TO 'phpmyadmin'@'localhost' IDENTIFIED BY 'Isi_Password_Nya_Disini';
Query OK, 0 rows affected (0.002 sec)

MariaDB [(none)]> FLUSH PRIVILEGES;
Query OK, 0 rows affected (0.001 sec)

MariaDB [(none)]> exit
Bye
[root@hamim-centos8 ~]#

Langkah terakhir restart apache dan database Anda

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# systemctl restart httpd
[root@hamim-centos8 ~]# systemctl restart mysql

Silakan akses http://IP_VM/phpmyadmin atau domain yang sudah di arahkan ke IP VM Anda, hasilnya seperti berikut

Silakan login dengan menggunakan user root untuk login ke database Anda
