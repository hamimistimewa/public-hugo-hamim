Mengenal Dan Instalasi Web Server IIS Di Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

Microsoft salah satu sistem operasi yang berbeda dengan sistem operasi yang lain. Microsoft mempunyai web server sendiri yang di beri nama IIS Web Server kepanjangannya yaitu “Internet Information Services” .

Web Server IIS besutan Microsoft juga salah satu sekian banyaknya web server yang mendukung teknologi Microsoft seperti halnya ASP.NET.

Bukan hanya itu saja, Web Server IIS juga dapat menjalankan bahasa pemrograman lain seperti PHP, Python, Java dan bahasa lainnya.

Berikut ini beberapa fitur IIS: - Manajemen web farm yang terpusat - Delegated remote manajemen - Admin tool yang kokoh - Infrastruktur web yang mempunyai skalabilitas tinggi - Caching dan kompresi yang dinamis - Diagnostic tool yang lengkap - Perlindungan server yang ditingkatkan - Secure content publishing - Perlindungan akses - Mendukung ASP.NET atau PHP - Dan lainnya

Selengkapnya mengenai IIS dapat merujuk pada link berikut: https://www.iis.net/

Berikut ini cara instalasi web server IIS di Windows Server 2016

Silakan masuk ke server manager dan klik pada menu Manage >> Add Roles Anda Features

Pada tampilan di bawah silakan Klik Next

Lalu pilih Roles-based or feature-based Installation.

Pilih Server Anda

Scroll ke bawah pilih Web Server (IIS)

Ketika muncul tampilan seperti gambar di bawah ini, klik Add Features

Web Server IIS telah berhasil di pilih, klik Next

Lalu klik Next.

Lalu klik Next.

Pilih service (centang) yang ingin di aktifkan sesuai dengan keinginan, Next untuk lanjut ke tahapan berikutnya

Informasi mengenai Web Server IIS, Klik Install

Tunggu proses instalasi Web Server IIS sampai selesai yang membutuhkan waktu

Web Server IIS telah berhasil di install

Untuk tab konfigurasi Web Server IIS berada di menu Tools >> Internet Information Services (IIS) Manager
