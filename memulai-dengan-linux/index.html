Memulai Dengan Linux

Written by [Nur Hamim] with ♥ on July 6, 2019 in CENTOS 7 Administrator

Hai guyss, pada kesempatan kali ini saya akan sharing tentang LINUX, dan sesuai dengan judul tema pada kali ini nantinya kita akan mempelajari tentang apa itu Linux. Nah sebelum itu saya ingin cerita terlebih dahulu tentang pertama kali saya menggunakan linux hehehehe 🙂

Pada mulanya saya pribadi kenal Linux itu pada saat saya duduk di bangku SMK kelas 2, ya cuma sekedar tau saja hehehehe. Alhamdulillah-nya saat kelas 3 SMK saya ditugaskan untuk melaksanakan PKL (Praktek Kerja Lapangan), pada saat itu saya PKL di BLC Telkom Klaten yang berada di kota klaten Jawa tengah.

Nah di BLC inilah saya mengenal lebih jauh tentang linux, bahkan di BLC hukumnya wajib bagi semua siswa PKL memakai Linux lo. Awalnya sih lumayan bingung menggunakan linux, karena taunya cuma windows saja pada saat SMK kelas 1 – 2 hehehhe. Akhirnya lama – kelamaan ternyata sangat enak sekali menggunakan Linux bahkan sekarang sampai jatuh cinta pada linux #Asyekk wkwkwk alhamdulillah sampai sekarang masih istiqomah menggunakan Linux.

Linux itu menurut saya salah satu sistem operasi yang keren pakai (banget), udah free/ bebas lagi kurang apa coba, nikmat dah pokoknya pakai linux. Dengan linux saja dapat berkembang dan berfikir intinya nie otak gerak mikirin linux itu enak banget. Intinya linux tak seinstan mie goreng #ehh tak seinstan windows maksudnya, kenapa demikian ?

Salah satu contoh yang sederhana kalau di linux kita mau install aplikasi tinggal menjalankan sourcode atau perintah di terminal, atau bisa juga installasi langsung menggunakan paket yang telah disediakan. Beda dengan windows mah terlalu enak #ehh tinggal klak klik klak klik jadi lo, beuh, mantull Dah 🙁

Aneh nya kebanyakan orang banyak yang beranggapan dan bilang kalau linux itu susah, padahal menurut saya pribadi linux itu yang paling enak dan mantap untuk digunakan. Ya intinya happy banget menggunakan Linux Guyss. Oke langsung ke tema saja ya guyss,

Tau nggak sih apa itu Linux ?

Pada cerita sebelumnya saya banyak sekali bilang kata (linux) apa sih itu linux ? kok bisa – bisanya bilang enak emang gimana rasanya hihihihih. Jadi Linux itu sistem operasi, sama sepertihalnya windows, perbedaan yang mendasar windows itu bersifat closed sorce atau komersial atau berbayar (membeli license), sedangkan linux itu Free (bebas), bebas dalam artian bisa menjalankan program, mempelajari sourcode program yang ada di linuxnya sendiri, bahkan kita juga dapat mendistribusikan ulang lo guyss, bukan hanya itu guyss kita juga dapat meningkatkan program dan mempublikasikannya secara Free (bebas), beuh mantull kan guyss.

Nah pada tulisan kali ini saya tidak akan membahas detail tentang sejarah dari linux sendiri yaa, (banyak guyss wkwk) mending baca secara mandiri ya pada tautan berikut :

Sejarah linux Wikipedia or Belajar Linux Dot ORG

Di Linux itu berbeda dengan sistem operasi yang lain guys, dengan kebebasannya maka, sangat banyak orang membuat atau mendistribusikan varian – varian linux yang biasanya dikenal dengan Distro (Distribution Sistem Operasi),

Ada 3 varian linux yang sangat populer digunakan yaitu

  1. Debian
  2. RedHat
  3. Slackware

Nah dari varian distro di atas kemudia linux berkembang atau mempunyai banyak turunan di setiap variannya. Lihat pada contoh gambar dibawah ini

Nah gambar diatas sangat jelas sekali yaa, contohnya saja yang paling banyak orang gunakan kalau Varian Distro Debian turunannya ada Ubuntu , Linux Mint dll . Sedangkan RedHat ada CentOS dan Fedora dan sebenarnya masih banyak lagi distro – distro linux lainnya teman – teman dapat mengetahui berbagai macam distro linux pada tautan berikut : DISTROWATCH

Oke mungkin cukup sekian untuk pemahaman singkat mengenai Linux yaa, semoga bermanfaat.
