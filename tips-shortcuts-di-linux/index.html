Tips Shortcuts Di Linux

Pada kesempatan kali ini kita akan belajar tentang shortcuts yang berada di linux, sebelum itu apasih itu shortcuts ?

Shortcuts di linux salah satu tombol kombinasi yang dapat kita gunakan untuk melakukan sebuah eksekusi perintah yang kita inginkan. Contohnya apabila kita ingin membuka terminal Linux , dapat dilakukan dengan cara mengetikkan Ctrl + T .

Menggerakkan Cursor di Terminal Linux :

Ctrl + A : Digunakan untuk berpindah ke awal perintah Ctrl + E : Digunakan untuk berpindah ke akhir perintah Ctrl + XX : Digunakan untuk memblock dan berpindah sesuai dengan letak dimana kita berada Alt + F : Digunakan untuk berpindah ke depan satu kata di baris saat ini Anda berada Alt + B : Kebalikanya dari Alt + F

Manipulasi Text di Terminal Linux

Ctrl + U : Digunakan untuk mengubah/ cut karakter dan dimana kursor Anda berada Ctrl + K : Digunakan untuk memotong/ cut satu baris karakter dimana kursor Anda berada Ctrl + Y : Digunakan untuk paste hasil cut Alt + T : Digunakan untuk menukar dua kata terakhir sebelum kursor Alt + U : Digunakan untuk mengubah huruf kecil menjadi kapital perkata Alt + C : Digunakan untuk mengubah depan kata menjadi huruf kapital Alt + D : Digunakan untuk delete per kata Ctrl + T : Digunakan untuk membuka tab baru

Riwayat/Histori Akses Di Terminal Linux

Ctrl + R : Digunakan untuk melihat histori perintah sebelumnya yang pernah kita jalankan di terminal

Kontrol Di Terminal Linux

Ctrl + L : Digunakan untuk membersihkan screen/ layar Ctrl + S : Digunakan untuk menghentikan output proses yang ditampilkan di screen, tapi tidak menyebabkan proses terlebih berhenti ðŸ™‚ Ctrl + Q : Digunakan untuk melanjutkan output proses ditampilkan di screen, untuk memberhentikannya kembali menggunakan Ctrl + S

Spesial Karakter

Ctrl + H : Digunakan untuk menambah tab ke bawah masih dalam 1 screen Ctrl + J : Digunakan untuk menambah tab ke kanan masih dalam 1 screen

Oke itulah beberapa shortcuts yang sering sekali kita gunakan dalam menggunakan linux
