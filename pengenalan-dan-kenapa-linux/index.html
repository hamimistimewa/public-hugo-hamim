Pengenalan Dan Kenapa Linux?

Apa itu Linux ?

Menurut Wikipedia

Linux is a family of free and open-source software operating systems built around the Linux kernel. Typically, Linux is packaged in a form known as a Linux distribution (or distro for short) for both desktop and server use. The defining component of a Linux distribution is the Linux kernel, an operating system kernel first released on September 17, 1991, by Linus Torvalds. Many Linux distributions use the word “Linux” in their name. The Free Software Foundation uses the name GNU/Linux to refer to the operating system family, as well as specific distributions, to emphasize that most Linux distributions are not just the Linux kernel, and that they have in common not only the kernel, but also numerous utilities and libraries, a large proportion of which are from the GNU project

Sangat jelas ya dari rangkuman diatas, Linux adalah sistem operasi atau perangkat lunak open source (bebas dan terbuka) namun jangan salah mengartikan bebas disini tidak semua Linux gratis terdapat juga beberapa distribusi Linux yang berbayar.

Linux dikemas dalam bentuk distribusi Linux atau yang biasa dibilang dengan distro Linux untuk penggunaan Desktop maupun Server.

Komponen utama dari distribusi Linux adalah kernel Linux, kernel sistem operasi yang pertama kali dibuat oleh Linus Torvalds dan pertama kali di rilis pada 17 September 1991.

Pada tutorial sebelumnya saya juga telah membahas beberapa distro Linux yang dapat teman - teman lihat pada tautan berikut: Memulai Dengan Linux

Kenapa Linux ?

  • Lebih cepat, aman, stabil
    • Membantu pengembang dari seluruh dunia berkontribusi, bukan hanya satu perusahaan
  • Dapat di cusomisasi sesuai dengan kebutuhan bahkan bisa dikembangkan kembali.
  • Cocok untuk digunakan secara personal dan produksi
  • Komunitas nya tersebar di seluruh dunia salah satunya Indonesia
  • Masih banyak lagi lainnya.

Itu lah pengenalan dan kenapa harus menggunakan Linux versi saya.
