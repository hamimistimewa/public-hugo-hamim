Instalasi CMS WordPress di Cyber Panel

Written by [Nur Hamim] with â™¥ on July 6, 2019 in CONTROL PANEL

Hello Guyss, masih pada semangat yaa untuk belajar mengenai control panel, tutorial kali ini melanjutkan tutorial sebelumnya, dimana sebelumnya kita telah belajar bagaiamana cara membuat website di Cyber Panel .

Sesuai dengan judul tutorial kali ini kita akan menginstall CMS WordPress di control panel Cyber Panel.

Oke, langsung saja ke langkah â€“ langkah installasinya ya

Pertama, silakan login terlebih dahulu ke control panel cyber panel Anda.

Kedua, masuk kemenu Website >> List Website >> Klik pada menu launch , seperti pada gambar dibawah ini

Scroll kebawah terdapat menu fitur untuk instalasi aplikasi, secara default Cyber panel telah menyediakan CMS WordPress, jadi Anda tinggal klik pada menu WordPress with LSCache seperti pada gambar dibawah ini

Selanjutnya silakan mengisikan credential mulai dari title untuk wordpress Anda sampai username dan password untuk login ke admin wordpress Anda nantinya, selain itu Anda juga diminta untuk mengisikan letak direktori untuk menyimpan berkas wordpress, seperti pada gambar berikut, jika sudah diisi semua klik pada menu Install Now untuk memulai instalasi wordpress

Tunggu sampai instalasi wordpress selesai, jika sudah selesai akan tampail seperti berikut

Pada langkah ini Anda telah berhasil melakukan instalasi wordpress, silakan mencoba akses domain Anda. Berikut tampilan default wordpress

Untuk login ke admin wordpress silakan ketikkan nama domain /wp-admin atau nama domain /wp-login contohnya : https://infone.info/wp-login.php , isikan username dan password login admin wordpress Anda yang telah Anda input atau buat pada proses intalasi di atas sebelumnya.

Jika Anda berhasil login maka Anda akan diarahkan ke halam admin wordpress

Oke cukup simple yaa untuk instalasi CMS WordPress di Cyber Panel, sangat banyak manfaat yang didapatkan apabila kita memasang wordpress di control panel cyber panel yang menggunakan webserver litespeed mulai dari proses yang cepat dan tersedia pula plugin chache di wordpress khusus untuk webserver litespeed.
