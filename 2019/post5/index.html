Cara Melakukan Soft Dan Hard Reset Mikrotik RouterBoard

Written by [Nur Hamim] with â™¥ on July 5, 2019 in NETWORKING

Pada tutorial kali ini saya akan share bagaimana cara untuk melakukan soft dan hard reset pada mikrotik routherboard. Untuk melakukan reset pada mikrotik terdapat 2 cara yaitu ada Soft Reset dan Hard Reset, untuk istilah Soft Reset yaitu mengembalikan settingan atau konfigurasi ke mode default atau pabrikan, dengan menggunakan tools atau aplikasi, Mikrotik sudah tersedia tools Winbox. Soft Reset juga bisa menggunakan terminal di Mikrotik dengan perintah system resert-configuration , selain itu Anda akan diminta untuk Konfirmasi pilihan y/N Apabila ingin meresetnya ketik y, apabila tidak ketik N.

Reset Menggunakan Winbox

Pertama kita masuk ke winbox terlebih dahulu, setelah itu pilih menu System -> Reset Configuration -> ceklist No Default Configuration -> Reset Configuration

Jika sudah, selanjutnya klik yes untuk me-reboot routerboard,

Silakan tunggu sampai proses reset pada routerboard selesai. Apabila sudah selesai maka routerboard akan otomatis restart dan settingan yang ada di router sama seperti awal lagi.

Soft Reset Mikrotik menggunakan Terminal Mikrotik

Pertama kita masuk terlebih dahulu ke winbox, setelah itu pilih menu New Terminal -> ketikkan perintah -> system reset-configuration -> pilih yes. Tunggu Proses reset sampai selesai.

Baru saja kita sudah mempelajari bagaimana cara melakukan Soft Reset. Sekarang kita akan bahas tentang Hard Reset. Berbeda dengan Soft Reset kalau soft reset mereset dari dalam tampa menyentuh perangkat.

Berikut ini langkah - langkah Hard Reset, sebagai berikut:

  1. Pertama siapkan Routerboard yang mau di reset, disini kami menggunakan Mikrotik versi RB 941
  2. Pastikan keadaan kabel power tidak di tancapkan
  3. Menggunakan benda yang sekiranya bisa menjangka atau masuk ke lubang reset pada mikrotik . Biasanya kita menggunakan pulpen atau obeng (+) kecil
  4. Colokkan benda tersebut ke dalam lubang reset, tahan dan masukkan kabel power, tunggu sampai lampu ACT berkedip â€“ kedip, dan tahan beberapa detik (sekitar 10 detik) pada router â€“ router tertentu biasanya berbunyi beep 1x dam lampu ACT akan berhrnti. Setelah itu kita baru cabut pulpen tadi beserta kabel powernya.
  5. Setelah itu colokkan kembali kabel power dan cobalah masuk atau login ke mikrotik anda, dan chek apakah router router sudah ter reset.

Itu cara â€“ cara yang dapat Anda gunakan untuk melakukan reset pada router Mikrotik.
