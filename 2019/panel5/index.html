Mengatur DNS di Cyber Panel

Written by [Nur Hamim] with ♥ on July 6, 2019 in CONTROL PANEL

Tutorial kali ini merupakan kelanjutan dari tutorial – tutorial sebelumnya yaitu instalasi control panel Cyber Panel di CentOS 7.

Biasanya kebanyakan kalau kita telah menginstall control panel yang paling pertama kali di konfigurasi yaitu DNS, karena menurut saya DNS ini sangat penting sekali.

Contoh singkatnya saja kalau sebelumnya kita akses control panel panel menggunakan IP maka dengan melakukan konfigurasi DNS ini Anda dapat menggunakan nama domain Anda.

Selain itu Anda juga dapat mengelola domain yang Anda miliki langsung melalui control panel mulai dari penambahan record subdomain, mail, dan sebagainya.

Hal yang dibutuhkan dalam mengikuti tutorial kali ini yaitu domain, disini saya sebelumnya telah mempunyai domain, saya pribadi langganan domain di CloudKilat kenapa saya berlangganan domain di CloudKilat, yang pertama fitur dan Dasboardnya lengkap elegan pulaa enak dilihat dan mudah tentunya untuk digunakan.

Oke, mari kita lanjut ke proses konfigurasi DNS pada Cyber Panel.

Pertama, silakan login ke administrator kontrol panel Cyber Panel Anda.

Kedua, masuk ke menu DNS >> Create Name Server

Disini kita akan membuat Name server. Apa fungsi dari Name server ?

Name server itu pusat dari record DNS yang digunakan untuk menyimpan semua record DNS.

Selanjutnya silakan tambahkan domain Anda, disini nama domain saya yaitu infone.info . Selain itu buat name server berikut detailnya :

First Nameserver: ns1.infone.info IP Address: Alamat IP Server / Control Panel Cyber Panel Second Nameserver: ns2.infone.info IP Address: Alamat IP Server / Control Panel Cyber Panel

Jika sudah, silakan save dengan menekan tombol Create Nameserver, seperti pada gambar dibawah ini

Jika berhasil dibuat maka akan ada informasi bahwasannya Nameserver telah berhasil di buat, seperti gambar berikut

Selanjuntya, yaitu membuat Private Nameserver atau bisanya sering disebut dengan Gluer Record . Disini saya berlangganan domain di CloudKilat, bagi Anda yang berlangganan domain di CloudKilat bisa mengikuti step berikut,

  1. Login ke portal area CloudKilat (http://portal.cloudkilat.com/)
  2. Service >> Domain >> infone.info >> Manage
  3. Private Nameserver
  4. Mendaftarkan Glue record dengan cara menambahkan ns1 dan ns2 ke alamat IP Server atau Control Panel Cyber Panel lalu Save Changes . Seperti pada gambar dibawah ini.

Pastikan penambahan private Nameservernya berhasil, seperti pada gambar dibawah ini

Jika sudah, selanjutnya mengubah nameserver domain menjadi ns1.infone.info dan ns2.infone.info .

Untuk mengubahnya silakan pindah ke menu Nameserver >> klik pada menu Use custom nameserver (enter below) >> Isikan nameserver nya >> Changes Nameservers. Seperti pada gambar dibawah ini.

Apabila terdapat kendala pada saat pembuatan private nameserver domain, silakan menghubungi support CloudKilat di info@cloudkilat.com atau support@cloudkilat.com .

Bagi Anda yang mempunyai domain di provider lain selain CloudKilat silakan menghubungi pihak penyedia domain Anda. Untuk proses penambahan private Nameserver atau Glue Record.

Selanjutnya apabila perubahan NameServer telah dilakukan pada domain tersebut maka domain akan mengalami proses propagasi dan untuk proses propagasi membutuhkan waktu maksimal 48 jam. Cepat atau lambat pada proses propgasi domain bergantung dengan resolver ISP yang Anda gunakan.

Jika proses propagasi domain telah selesai, Anda dapat mengeceknya menggunakan command dig . Berikut contohnya :

$ dig @infone.info infone.info +short
117.53.46.27
$ dig @ns1.infone.info infone.info +short
117.53.46.27
$

Oke saat ini kedua nameserver telah resolve ke IP control Panel Cyber Panel. Selanjutnya Anda dapat menambahkan A record atau record DNS yang lainnya sepertihalnya MX record dan yang lainnnya melalui DNS Management Control Panel Cyber Panel di menu DNS >> Add/Delete Records . Seperti pada gambar dibawah ini

Untuk mencobanya membutikan apakah domain infone.info telah mengarah ke control panel Cyber Panel dapat dilakukan dengan cara mengakses Control Panel Cyber Panel mengguakan nama domainnya diikuti port cyber panel contohnya : https://infone.info:8090

Atau Anda dapat melakukan ping ke domain infone.info , maka hasilnya domain tersebut dapat di ping dan mengarah ke IP control Panel Plesk Panel Anda :

root@hamimaja:~# ping infone.info

Oke, saat ini konfigurasi DNS pada Cyber Panel telah berhasil, selamat mencoba semoga bermanfaat 🙂
