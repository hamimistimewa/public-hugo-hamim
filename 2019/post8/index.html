NetInstall Mikrotik

Written by [Nur Hamim] with ♥ on July 6, 2019 in NETWORKING

Apa itu Netinstall ?

Netinstall merupakan program yang berjalan pada komputer Windows yang memungkinkan Anda untuk menginstall Mikrotik RouterOS ke Komputer atau ke RouterBoard melalu jaringan Ethernet. Netinstall program berbasis windows dengan protokol Bootp. Netinstall juga bisa di jalankan di Sistem Operasi Linux dengan emulator wine, Netinstall juga bisa dijalankan di Linux.

Netinstall digunakan pada saat:

  1. Instalasi sebelumnya gagal
  2. OS dalam Router rusak
  3. Kehilangan Password untuk akses
  4. Di gunakan untuk mereset password atau lupa password

Yang Harus kita persiapkan untuk menjalankan netinstall adalah :

  1. Program Netinstall
  2. Kabel UTP
  3. File software Mikrotik sesuai dengan jenis nya : x86, mipsbe, mipsle, dan sejenisnya.

Untuk mendownload Netinstall Anda bisa mendownloadnya di situs resmi Mikrotik (www.mikrotik.com/download)

Silakan download Package RouterOS, sesuaikan dengan tipe router Anda dan versi yang ingin Anda gunakan jika menggunakan RB750 silahkan pilih tipe MIPSBE, jika menggunakan RB941-2nD – TC silahkan pilih tipe SMIPS. Di sini kami akan menggunakan RB941 jadi kami mendownloadnya yang SMIPS dari halaman https://www.mikrotik.com/download

Berikut ini cara menggunakan Netinstall:

Setting IP Address dan Gateway di Komputer kita

  • IP Adress : 192.168.2.2 dan Gateway 192.168.2.1

  • Lalu klik OK

  • Setting IP Address di Linux Ubuntu

  • Setting IP Address di Windows

  • Setelah itu masuk ke Applikasi Netinstall >> Klik NetBooting -> Ceklis Boot Server Enable >> Isikan IP Address Client : 192.168.2.1 (IP Gateway Komputer)

  • Cabut Kabel Power dari Routerboard
  • Reset Routerboad menggunakn pulpen atau sejenisnya, tahan sekitar 10 detik
  • Colokan kembali kabel power ke RouterBoard
  • Tunggu Sampai Router kita terdeteksi oleh Netinstall tunggu sekitar 15 detik.
  • Kita klik browser
  • Lalu pilih Package yang akan di install sesuai dengan tipe dan versi Routerboard
  • Lalu klik OK

  • Selanjutnya klik select all untuk menandai semua package
  • Lalu klik install

  • Setelah itu kita tunggu sampai instalasi OS di Routerboard selesai.

  • Pastikan installasi selesai seperti pada gambar di bawah ini

Jika sudah selesai maka Routerboard akan kembali ke settingan default.

Selamat mencoba semoga bermanfaat 🙂
