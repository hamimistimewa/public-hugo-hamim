Oke saya harap santai sejenak yaa jangan tegang hehehehe, pada tutorial kali saya akan bahas tuntas mengenai backup keselururahan isi WordPress mulai dari plugin sampai file â€“ file/ data yang ada di WordPress kita ke S3 Protokol.

Disini saya menggunakan S3 Protokol dari CloudKilat untuk penyimpanan backupnya, kenapa saya memilih S3 Protokol CloudKilat ?

Alasan yang paling utamanya adalah tempat atau letak layanan tersebut berada di Indonesia tepatnya di Data Center Cyber 2 Jakarta selain itu harga layanan yang bisa dibilang terjangkau disertai kualitas layanan yang mantull #BukanKaleng-Kaleng #ehh

Nama layanannya yaitu Kilat Storage, pertama kali saya makai ni untuk media penyimpanan sangatlah memuaskan haaahhaahah kenapa ?

Karena jarang dan bahkan nggak pernah ada downtime, dengan demikian earningku jalan dengan lancar, matursuwon CloudKilat hehehe ðŸ™‚

Kedua, silakan install dan aktifkan terlebih dahulu plugin UpdraftPlus dengan cara berikut : Login Admin WordPress >> Plugins >> Add new >> Search nama Plugin Updraftplus >> Install Now >> Activate

Apabila plugin telah berhasil diintall, berikutnya membuat bucket atau tempat penyimpanan backup WordPress. Untuk bucketnya dibuat di layanan Kilat Storage. Untuk mengakses layanan Kilat Storage sangat banyak cara mulai dari menggunakan Dragondisk, s3cmd, s3browser dan masih banyak yang lainnya. Untuk detailnya bisa di lihta di tautan berikut :

Cara Mengakses layanan Kilat Storage CloudKilat

Disini saya menggunakan dragondisk, terlebih dahulu silakan buka portal area CloudKilat Anda untuk melihat Akses Key dan Secret Key dari Layanan Kilat Storage Anda, dengan cara Login Portal CloudKilat >> Service >> Kilat Storage >> View, scroll kebawah Anda akan mendapatkan informasi detailnya seperti pada gambar berikut

Jika Anda telah mengetahui Access Key dan Secret Key layanan Kilat Storage Anda silakan buka aplikasi Dragondisk untuk membuat bucket dengan cara

Buka Aplikasi Dragondisk >> Account >> New >> Isikan detail informasi seperti Endpoint, secret key dan access key yang dibutuhkan seperti berikut

Apabila Anda telah membuat bucket, silakan kembali lagi ke plugin wordpress, dimana kita akan melakukan konfigurasi, silakan masuk ke plugin dengan cara setting >> UpdraftPlusBackups >> settings secara default seperti berikut, silakan pilih yang menu S3 Compatible (Generic)

Anda dapat menentukan jadwal agenda backup yang Anda inginkan mulai dari, monthly, fotnighly, weekly, daily, Every 12 hours, dll .

Gambar diatas kita juga mengisikan credential mulai dair endpoint Kilat Storage, nama bucket, access key, secret key.

Untuk testing apakah backup kita berhasil tersimpan ke penyimpanan Kilat Storage mari kita coba backup manual dulu, dengan cara pindah ke menu Backup/Restore >> Backup Now maka si plugin akan membuat backup wordpress mulai dari Backup , plugin, database, upload dll. Jika backupnya udah berhasil Anda akan mendapatkan hasil backup tersebut seperti pada gambar dibawah ini.
