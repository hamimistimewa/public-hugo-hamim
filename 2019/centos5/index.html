Alias Login SSH di Linux

Written by [Nur Hamim] with ♥ on July 6, 2019 in CENTOS 7 Administrator

Pada kesempatan kali ini topik yang akan kita bahas masih sama yaitu tentang SSH, sebelumnya kita telah mempelajari bagaimana cara melihat log di SSH.

Nah kali ini kita akan belajar cara bagaimana cara membuat alias atau modifikasi untuk login ssh di linux.

Manfaat yang didatpakan yaitu Anda dapat mempermudah akses SSH dengan menggunakana nama yang Anda inginkan tanpa perlu mengetikkan perintah ssh roo@IP_Adress kembali.

Oke langsung saja yaa ke tahapan – tahapan membuat alias untuk login SSH di Linux.

Pertama, membuat sebuah file konfigurasi untuk meyimpan aliases ssh , di direktori /home/namauser/.ssh/ seperti berikut :

$ sudo vim ~/.ssh/config

Kedua, isikan file config tersebut seperti pada gambar berikut :

Keterangan :

  • Host : Nama yang ingin dipanggil atau yang ingin digunakan untuk mengakses ssh
  • User : Merupakan user ssh, gambar diatas menggunakan root . Untuk user bersifat bebas menyesuaikan dengan user ssh Anda.
  • Hostname : Merupakan alamat dari server yang ingin kita remote
  • Port : Isikan port ssh Anda.
  • IdentityFile : sama seperti port, namun disini digunakan apabila Anda menggunakan ssh key.

Apabila Anda telah membuat konfigurasinya, silakan simpan konfigurasi tersebut lalu coba akses ssh Anda menggunakan Host . Berikut hasilnya :

Itulah cara singkat bagi kita untuk menggunakan ssh secara praktis.
