Membuat Website di Cyber Panel

Written by [Nur Hamim] with â™¥ on July 6, 2019 in CONTROL PANEL

Hai guyss untuk tutorial kali ini saya akan share bagaimana cara membuat sebuah website menggunakan control panel cyber panel, tutorial ini kelanjutan dari tutorial sebelumnya yang memabahas tentang bagaimana cara menngatur DNS di Cyber Panel

Untuk mengikuti tutorial ini pastikan Anda telah mengatur DNS dengan benar, jika sudah mari kita bahas bagaimana cara membuat website di cyber panel

Pertama, login terlebih dahulu ke cyber panel Anda

Kedua, pilih menu Website >> Create Website , dan isikan detailnya mulai dari pemilihan paket yang ingin digunakan, disini saya menggunakan packages default.

Selain itu Anda juga dapat menentukan apakah website nantinya ingin menggunakan SSL Free yang disediakan oleh Cyber Panel atau tidak, jika Anda ingin menggunakan SSL Free silakan checklist pada menu SSL yang berada di Additional Features seperti pada gambar dibawah ini

Tunggu proses pembuatan website sampai selesa

Jika sudah selesai maka Akan ada informasi seperti pada gambar berikut

Untuk melihat list dari website yang telah kita buat dapat dilihat dimenu Website >> List Website . Klik pada icon Launch untuk melihat fitur apa saja yang dapat diinstall atau dipasangkan di website Anda nantinya.

Jika sudah di klik, Anda akan dimanjakan oleh tampilan fitur yang diberikan oleh Cyber panel yang sangat â€“ sangat indah dan mudah dalam penggunaanya. Nggak ribet uyyy :

Silakan Akses website yang telah Anda buat, seperti berikut hasilnya, website kita telah terinstall SSL Free looo.
