Cara Konfigurasi OpenVPN Server dan Client Pada pfSense 2.4

Written by [Nur Hamim] with â™¥ on February 18, 2020 in Sinau pfSense 2.4

Mengenal Apa Itu VPN

VPN salah satu metode yang dapat kita gunakan untuk menghubungkan suatu jaringan dengan jaringan lainnya melalui internet secara aman.

OpenVPN merupakan aplikasi yang dapat digunakan untuk membuat membuat VPN Server dan Client OpenVPN juga salah satu aplikasi yang sering dan banyak sekali digunakan.

OpenVPN sendiri terdapat 2 versi yaitu:

  1. OpenVPN Community Edition
  2. OpenVPN Access Server

Perbedaan dari versi diatas diaantaranya:

  1. OpenVPN Community Edition: Untuk versi ini semua konfigurasi di harus dilakukan melalui command line atau console
  2. OpenVPN Access Server: Untuk versi ini sudah terdapat web GUI yang memudahkan administrator baik dari instalasi sampai dengan management VPN.

Kedua versi tersebut free/gratis, namun untuk Access Server, secara default terdapat limit 2 concurrent user. Agar bisa digunakan lebih dari 2 user, kita harus membeli license tambahan.

Untuk harganya dapat melihat pada link berikut: https://openvpn.net/index.php/access-server/pricing.html.

Namun pada tutorial kali ini kita akan membuat server OpenVPN menggunakan pfSense, pada pfSense OpenVPN sudah menjadi service yang secara default sudah ada, Anda hanya perlu mengkonfigurasinya saja.

Tahapan Konfigurasi OpenVPN Server

Langkah pertama kita akan membuat CA_ROOT dan Certificates, silakan masuk ke menu System >> Cert. Manager

Kemudian Anda ke menu CAs lalu Add

Membuat CA_ROOT untuk certificates OpenVPN, silakan isi yang perlu saja mulai dari Descriptive name, Common Name, Country Code, State of Province, City, Organization, kemudian Save seperti berikut

Hasilnya akan seperti berikut ini

Langkah selanjutnya pindah ke menu Certificates >> Add

Silakan isi seperti membuat CA_ROOT sebelumnya, sesuaikan saja dengan keinginan untuk Descriptive name, Common Name, gambarannya seperti berikut

Hasilnya akan seperti ini

Langkah selanjutnya konfigurasi OpenVPN Server silakan masuk ke menu VPN >> OpenVPN

Pindah ke menu Wizard

Pilih Local User Access, lalu Next

Pilih CA yang telah dibuat sebelumnya, lalu Next

Pilih Certificates yang telah dibuat sebelumnya, lalu Next

Pada menu General OpenVPN Server Information silakan isikan Description sesuai keinginan Anda

Pada menu Cryptographic Settings pastikan untuk Auth Digest Algorithm menggunakan **SHA256 (256-bit)

Pada menu Tunnel Network, Redirect Gateway, Local Network, Concurrent Connections, silakan isikan sesuai keinginan,

Pada menu Client Settings silakan isi DNS Default Domain dengan hostname pfsense Anda dan untuk DNS Server Anda dapat menggunakan DNS Resolver Google, lalu Next untuk melanjutkannya seperti berikut

Langkah selanjutnya pada ceklis pada menu Firewall Rule & OpenVPN rule

Langkah selanjutnya klik Finish

Berikut hasilnya

Pastikan pada Firewall >> Rules sudah ada rules untuk OpenVPN

Tahapan Konfigurasi OpenVPN Client

Langkah selanjutnya melakukan konfigurasi OpenVPN Client silakan pindah ke menu Clients >> Add

Pada menu General Information silakan isikan IP atau hostname pfsense

Pada menu User Authentication Settings Anda dapat membuat credential untuk OpenVPN Client, harap dicatat ini bukan untuk user OpenVPN

Pada menu Cryptographic Settings silakan tambahkan NCP Algorithms seperti pada gambar berikut

Pada menu Advanced Configuration silakan pilih IPv4 Only

Instalasi Paket OpenVPN

Paket OpenVPN disini digunakan untuk mengunduh user vpn sesuai device yang ingin kita gunakan, berikut tahapan - tahapannya:

Silakan masuk ke menu System >> Package Manager

Lalu search openvpn >> Install

Lalu pilih Confirm untuk melanjutkannya

Tunggu proses instalasi sampai selesai seperti berikut

Membuat User OpenVPN

Silakan masuk ke menu System >> User Manager

Pada menu User >> Add

Isikan dengan detail sebagai berikut:

  • Username: isikan username OpenVPN (sesuai kebutuhan)
  • Password: Isikan password dari user diatas
  • Descriptive name: Isikan sesuai keinginan Anda

Jika sudah silakan di Save

Berikut hasil dari user OpenVPN yang telah kita buat

Untuk mengunduh file openvpn sesuai user yang telah kita buat sebelumnya silakan masuk ke menu VPN >> OpenVPN >> Client Export

Scroll ke bawah Anda akan menemukan menu OpenVPN Clients silakan unduh sesuai device yang Anda gunakan
