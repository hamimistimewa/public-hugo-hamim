Cara Menambahkan Volume Di Windows Server 2016

Written by [Nur Hamim] with â™¥ on August 21, 2019 in Windows Server 2016

Menambah storage di sebuah sistem operasi merupakan suatu hal yang wajib kita ketahui, dengan kita mengetahui bagaimana cara menambahkan sebuah disk maka kita akan mudah kedepannya bila terdapat issue atau case tentang storage misalnya penambahan storage baru yang akan digunakan untuk menyimpan data atau penambahan storage baru yang ingin digunakan untuk menyimpan aplikasi aplikasi tertentu yang membutuhkan penyimpanan khusus.

Berikut ini merupakan tahapan bagaimana cara menambahkan storage di Windows Server 2016.

Pada gambar dibawah terlihat Disk yang ada saat ini di Windows Server saya hanyalah Disk root (C:)

Untuk menambahkan storage baru misalnya Disk (D: , E:) dapat dilakukan.

Biasanya setelah instalasi Windows Server kita perlu mengaktifkan Disk (D: , E:) secara manual.

Perlu di ketahui juga bila Anda menggunakan VPS atau Virtual Machine di provider/penyedia layanan Cloud Computing silakan menghubungi pihak penyedia Cloud Computing Anda untuk melakukan Attack Volume ke VPS / Virtual Machine Anda jika sudah Anda dapat menambahkannya secara mandiri sesuai dengan keinginan di sistem operasi Windows Server Anda.

Silakan akses Windows Server 2016 Anda masing - masing, dan mengakses Server Manager

Berikutnya klik pada menu File anda Storage Service

Kemudian klik pada menu Disk dan klik pada spot yang kosong dan klik “Rescan Storage”

Pada gambar di atas sudah terlihat disk baru yang masih “Unknown” silakan klik kanan lalu “Bring Online” . Bring Online digunakan untuk mengaktifkan disk

Selanjutnya Anda dapat membuat Volume dengan cara klik kanan pada disk >> New Volume

Akan muncul konfigurasi Volume, klik Next

Pada menu “Server and Disk” sudah terlihat Disk yang baru saja di aktifkan, klik Next

Selanjutnya menentukan size dari Volume, disini saya akan menambahkan volume sebesar 50GB, klik Next

Selanjutnya pada menu “Drive Letter or Folder” Anda dapat menentukan Drive Letter ingin di taruh di Disk E etc, seperti pada gambar di bawah ini, klik Next

Pada tahap ini Anda diminta untuk memilih file system yang ingin digukana Anda bisa menggunakan NTFS atau ReFS, selain itu Anda juga dapat memberikan nama atau label pada Volume Anda seperti pada gambar di bawah ini

Kemudian pada menu “Confirmation” Klik Create untuk membuat Volume, dan silakan tunggu sampai complete

Saat ini Disk Volume sudah berhasil di tambahkan.
