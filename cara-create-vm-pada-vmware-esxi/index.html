Cara Create VM Pada VMware ESXi

Written by [Nur Hamim] with ♥ on February 16, 2020 in Sinau pfSense 2.4

Pada tutorial kali ini kami akan memberikan cara bagaimana membuat VM pada VMware ESXi, dimana nantinya VM ini akan digunakan untuk instalasi pfSense 2.4 dengan spesifikasi yang dibutuhkan sebagai berikut:

  • CPU : 2 Core
  • RAM : 2 GB
  • HDD : 10 GB
  • Network : 2 interface

Requirements hardware yang dibutuhkan untuk instalasi pfsense dapat dilihat pada link berikut: Minimum Hardware Requirements.

Selanjutnya, silakan akses VMWare Anda masing - masing dan create VM

Kemudian pilih custom, lalu Next

Memberi nama VM, silakan sesuaikan dengan keinginan masing - masing

Selanjutnya, memilih Datastore yang ingin digunakan untuk menampung VM pfsense nantinya.

Selanjutnya memilih kompabilitas pilih “Virtual Machine Version 11”

Pada menu Gest OS ini kita akan menggunakan Free-BSD 64 Bit, karena pfSense sendiri merupakan turunan dari Free-BSD.

Selanjutnya menentukan CPU yang ingin digunakan, sesuai info sebelumnya kita akan menggunakan 2 core CPU

Menentukan RAM atau memory yang digunakan kali ini yaitu sebesar 2 GB

Pada menu Network kita akan menggunakan 2 network dengan satu Interface yang sama. 1 Network digunakan sebagain WAN Network (Internet-Vlan) dan 1 Network digunakan untuk LAN Network (Local-Vlan) dengan type Interface VMXNET3

Selanjutnya untuk disk akan menggunakan disk baru silakan pilih Create a new virtual disk, lalu Next

Sesuai info sebelumnya untuk disk yang kita gunakan sebesar 10 GB

Pada menu Advanced Options pada VDN pilih SCSI lalu Next

Langkah berikutnya klik Finish

Langkah Selanjutnya klik kanan pada VM >> Edit Settings >> Pilih CD/DVD >> Pastikan sudah menceklis Connect at power on >> upload ISO pfSense 2.4.4 yang sudah di unggah ke Datastore sebelumnnya.

Langkah terakhir silakan power on VM
