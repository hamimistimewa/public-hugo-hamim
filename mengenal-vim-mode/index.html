Nah jenis â€“ jenis mode di vim bisa di bilang banyak diantaranya :

Mode Normal

Digunakan untuk navigasi kursor dan memanipulasi teks. Mode ini yang pertama kali dimasuki saat pertama kali menjalankan vim.

Mode Insert

Digunakan untuk menambah teks baru

Mode Visual

Digunakan untuk navigsi kursor dan manipulasi teks secara visual dengan menggunakan text block.

Mode Command

Digunakan untuk menjalankan perintah yang dikenal oleh Vim.
