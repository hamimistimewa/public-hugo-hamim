Cara Instalasi PHP 7.2, 7.3 dan 7.4 Apache Di CentOS 8

Written by [Nur Hamim] with â™¥ on March 12, 2020 in CentOS 8 Administrator

Pengenalan

PHP adalah salah satu bahasa pemrograman yang paling banyak digunakan. Banyak CMS dan Frameworks seperti WordPress, Magento, dan lavarel dibangun menggunakan PHP.

Instalasi PHP

CentOS 8 didistribusikan dengan default PHP versi 7.2. Versi sebagian besar sudah mendukung aplikasi PHP modern contohnya seperti WordPress yang sudah wajib menggunakan PHP 7.x. Versi PHP yang terbaru ataupun up to date tersedia di Remi Repository.

Instalasi Repository Remi

Jika Anda akan menggunakan PHP versi 7.2, maka langkah ini bisa Anda lewatkan. Namun jika Anda nantinya akan menggunakan PHP 7.3 dan 7.4, maka Anda perlu instalasi repository remi terlebih dahulu, berikut perintah yang digunakan untuk installasinya

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# dnf install dnf-utils http://rpms.remirepo.net/enterprise/remi-release-8.rpm

Setelah instalasi selesai dilakukan, untuk melihat semua versi yang tersedia jalankan perintah berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# dnf module list php
Last metadata expiration check: 0:07:07 ago on Wed Mar 11 05:31:05 2020.

php                                7.3                                   common, devel, minimal                                  PHP scripting language

Hint: [d]efault, [e]nabled, [x]disabled, [i]nstalled
[root@hamim-centos8 ~]#

Seperti yang disampaikan diatas sebelumnya secara default di CentOS 8 sudah diatur menggunakan PHP 7.2 dengan tanda [d] diatas, silaka sesuaikan dengan kebutuhan masing - masing.

Jika kebutuhannya menggunakan PHP 7.2 Anda dapat langsung melakukan installasi PHP nya, namun jika Anda ingin menggunakan PHP versi 7.3 atau 7.4 Anda dapat mengubah set module nya dengan menggunakan beberapa perintah berikut

PHP 7.3

[root@hamim-centos8 ~]# dnf module reset php
[root@hamim-centos8 ~]# dnf module enable php:remi-7.3

PHP 7.4

[root@hamim-centos8 ~]# dnf module reset php
[root@hamim-centos8 ~]# dnf module enable php:remi-7.4

Disini kami akan menggunakan PHP versi 7.3, seperti berikut hasilnya

Jika sudah Anda dapat install extensi atau module PHP yang dibutuhkan, silakan sesuaikan dengan aplikasi Anda masing - masing, sebagai contoh berikut kami berikan cara instalasi beberapa module PHP nya

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# dnf install php php-opcache php-gd php-curl php-mysqlnd php-fpm -y

Jika ingin mencari extension yang dibutuhkan bisa menggunakan perintah berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# dnf search php-

Untuk melihat versi PHP yang sudah diinstall dapat menggunakan perintah berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# php -v
PHP 7.3.15 (cli) (built: Feb 18 2020 09:25:23) ( NTS )
Copyright (c) 1997-2018 The PHP Group
Zend Engine v3.3.15, Copyright (c) 1998-2018 Zend Technologies
    with Zend OPcache v7.3.15, Copyright (c) 1999-2018, by Zend Technologies
[root@hamim-centos8 ~]#

Untuk melihat info.php gunakan perintah berikut

[root@hamim-centos8 ~]#
[root@hamim-centos8 ~]# echo '<?php phpinfo(); ?>' > /var/www/html/info.php

Langkah terakhir restart web server apache Anda

[root@hamim-centos8 ~]# systemctl restart httpd
[root@hamim-centos8 ~]#

Jika sudah silakan akses http://IP_VM/info.php atau http://namadomain.com/info.php contohnya https://vhost.nurhamim.net/info.php hasilnya sebagai berikut
