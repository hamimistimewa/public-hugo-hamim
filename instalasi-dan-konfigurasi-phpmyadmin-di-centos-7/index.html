Instalasi Dan Konfigurasi phpMyAdmin Di CentOS 7

Pada tutorial tersebut kita juga dapat membuat database menggunakan mode CLI (Command Line Interface) nah untuk mempermudah Anda dalam pengelolaan Database Anda dapat menggunakan phpMyAdmin dimana phpMyAdmin sendiri berbasis Web Based.

Dimana yang dimaksuda database Anda dapat diakses melalui web browser dengan menggunakan phpMyAdmin.

Pada tutorial kali ini kita akan menginstall phpMyAdmin menggunakan webserver apache bagi yang belum install webserver apache silakan install terlebih dahulu.

Pertama, silakan akses server CentOS 7 Anda masing â€“ masing,

Kedua, pastikan Anda telah menginstall database server MariaDB atau MySQL dan webserver apache

Ketiga, silakan melakukan pembaharuan atau update sistem operasi CentOS 7 Anda.

Berikutnya, menginstall atau menambahkan repository epel di CentOS 7 dengan cara menjalakan perintah berikut

[root@srv-hamim01 ~]# yum install epel-release

Jika suda, silakan install phpMyAdmin di CentOS 7 dengan cara menjalankan perintah berikut

[root@srv-hamim01 ~]# yum install phpmyadmin

Silakan tunggu proses instalasi sampai selesai, jika sudah selesai proses installasinya coba akses alamat IP server Anda /phpmyadmin

Santai setelah installasi kita perlu melakukan konfigurasi phpMyAdmin, untuk file yang harus kita konfigurasi sendiri berada di /etc/httpd/conf.d/phpMyAdmin.conf . Untuk kebutuhan edit file konfigurasi dapat menggunakan editor vim atau nano, sesuai dengan keinginan Anda masing â€“ masing, disini penulis menggunakan vim

[root@srv-hamim01 ~]# vim /etc/httpd/conf.d/phpMyAdmin.conf

Berikut isi dari file phpMyAdmin secara default

Silakan menambahkan perintah Require all granted dan disabled ip localhost seperti pada gambar dibawah ini

Jika sudah, silakan simpan file phpMyAdmin lalu melakukan reload atau restart terhadap webserver apache kita dengan cara menjalankan perintah berikut

[root@srv-hamim01 ~]# systemctl restart httpd

Jika, sudah silakan akses kembali IP server disertai /phpmyadmin, insya Allah work 100% ganss ehhehehe, hasil seperti berikut ini

Untuk loginnya secara default masih menggunakan root apabila Anda belum membuat user baru, untuk passwordnya password root database yang telah Anda buat pada saat Anda installasi database sebelumnya.
