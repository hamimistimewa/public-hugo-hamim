Instalasi pfSense 2.4

Written by [Nur Hamim] with â™¥ on February 16, 2020 in Sinau pfSense 2.4

Tutorial kali ini merupakan lanjutan dari tutorial sebelumnya, dimana sebelumnya kita telah create VM dan pada tutorial kali ini kita akan melakukan instalasi pfsense 2.4 namun sebelum melakukan instalasi kita akan membahas sedikit tentang pfSense.

Mengenal pfSense

pfSense adalah software (operating system) open source berbasis FreeBSD (unix) yang diperuntukan atau digunakan sebagai firewall sekaligus router, pfSense dapat diinstall di server fisik (on-premise) atau di virtual machine atau compute di Cloud Computing.

Untuk melakukan managemen pfSense menggunakan interface web base yang tentunya akan memudahkan kita dalam melakukan management. pfSense mempunyai banyak fitur seperti VPN Server, Free Radius, HAProxy, Proxy Server dan selengkapnya untuk melihat secara detail fitur apa saja yang disediakan silakan klik pada link berikut: pfSense Features.

pfSense didistribusikan dibawah license Apache 2.0 & ter-Copyright 2004-2018 oleh Rubicon Communications, LLC (Netgate).

Jika Anda ingin mengunduh pfsense dapat melalui situs resmi pfsense pada link berikut: pfSense Donwload.

Instalasi pfSense 2.4

Pertama - tama yang harus Anda lakukan yaitu melakukan power on pada VM dan silakan login ke console VM untuk melanjutkan instalasi pfsense

Pada langkah ini silakan Install pfSense lalu OK

Selanjutnya pilih Continue with default keymap lalu pilih Select

Pada pemilihan partisi disini kita menggunakan auto silakan pilih Guided Disk Setup

Tunggu proses instalasi sampai selesai yang membutuhkan waktu

Jika instalasi sudah selesai Anda akan mendapatkan opsi untuk membuka shell jika terdapat konfigurasi tambahan, disini pilih No

Selanjutnya melakukan reboot

Tunggu proses booting sampai selesai

Melakukan konfigurasi pada pfSense, disini terdapat opsi apakah Anda ingin melakukan konfigurasi Vlan, ketik n lalu Enter

Set name interface yang digunakan untuk oleh WAN Interface

Set name interface yang akan digunakan oleh LAN Interface

Ketikkan y untuk menyimpan konfigurasi yang dilakukan sebelumnya

Langkah selanjutnya yaitu melakukan konfigurasi IP Public pada Interface WAN (vmx0), silakan pilih opsi 2 untuk konfigurasi IP Address

Lalu pilih opsi 1 (Interface WAN) yang ingin di berikan IP Address static

Selanjutnya silakan konfigurasi IP Adress seperti berikut, masukkan IP Address Public, Subnetmask dan IP Gateway.

Jika Anda tidak ingin mengaktifkan IPv6 silakan di disable saja ketikan N

Selanutnya pada kofigurasi webconfigurator pilih opsi y dan tunggu hingga konfigurasinya selesai

Sebelum mengakses pfsense melalui web browser silakan masuk ke shell dan jalankan perintah berikut pfctl -d

Setelah itu akses IP Public VM yang sudah ditentukan sebelumnya pada interface WAN, untuk detail credential default pfsense sebagai berikut:

Username: admin Password: pfsense

Instalasi pfsense telah selesai dilakukan.
